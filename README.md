# Imbrace SDK Monorepo

TypeScript and Python SDKs for the Imbrace Gateway.

| Directory | Package | Description |
| --- | --- | --- |
| [`ts/`](./ts) | `@imbrace/sdk` | TypeScript/JavaScript — Node.js and browser |
| [`py/`](./py) | `imbrace` | Python — sync and async clients |

---

## Setup

### TypeScript
```bash
cd ts
npm install
npm run build
```

### Python
```bash
cd py
pip install -e ".[dev]"
```
