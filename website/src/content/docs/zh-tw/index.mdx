## 為什麼選擇 Imbrace SDK？

<CardGrid>
  <Card title="型別安全設計" icon="approve-check-circle">
    在所有 27+ 個資源命名空間提供完整的 TypeScript 定義和 Python 型別提示。在編譯期而非執行期捕捉錯誤。
  </Card>
  <Card title="自動重試與韌性" icon="random">
    內建針對 429 和 5xx 回應的指數退避。無需任何設定 — 您的應用程式在負載下依然正常運作。
  </Card>
  <Card title="三層認證" icon="setting">
    API 金鑰、JWT Bearer 及舊版存取令牌 — 全部透明處理。在不改變商業邏輯的情況下切換認證策略。
  </Card>
  <Card title="Python 優先非同步" icon="rocket">
    並行 `AsyncImbraceClient` 與同步客戶端並存。一次匯入即可在 FastAPI、asyncio 或 Django 中使用。
  </Card>
</CardGrid>

</div>

## 快速安裝

<div class="install-grid">
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--ts">TypeScript</span>
    </div>

    ```bash
    npm install @imbrace/sdk
    ```

    ```ts
    import { ImbraceClient } from "@imbrace/sdk";
    const client = new ImbraceClient();
    const me = await client.platform.getMe();
    ```

  </div>
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--py">Python</span>
    </div>

    ```bash
    pip install imbrace
    ```

    ```python
    from imbrace import ImbraceClient
    client = ImbraceClient()
    me = client.platform.get_me()
    ```

  </div>
</div>
