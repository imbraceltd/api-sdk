## 為什麼選擇 Imbrace SDK？

<CardGrid>
  <Card title="類型安全設計" icon="approve-check-circle">
    覆蓋 27+ 資源命名空間的完整 TypeScript 定義和 Python 型別提示。在編譯時捕獲錯誤，而非執行時。
  </Card>
  <Card title="自動重試與彈性" icon="random">
    針對 429 和 5xx 回應內建指數退避重試。無需配置 — 高負載下應用依然穩定運行。
  </Card>
  <Card title="三層身份驗證" icon="setting">
    API Key、JWT Bearer 和 legacy access token — 全部透明處理。無需更改業務邏輯即可切換認證策略。
  </Card>
  <Card title="Python 非同步優先" icon="rocket">
    `AsyncImbraceClient` 與同步客戶端並行存在。一行匯入即可接入 FastAPI、asyncio 或 Django。
  </Card>
</CardGrid>

</div>

## 快速安裝

<div class="install-grid">
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--ts">TypeScript</span>
    </div>

    ```bash
    npm install @imbrace/sdk
    ```

    ```ts
    import { ImbraceClient } from "@imbrace/sdk";
    const client = new ImbraceClient();
    const me = await client.platform.getMe();
    ```

  </div>
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--py">Python</span>
    </div>

    ```bash
    pip install imbrace
    ```

    ```python
    from imbrace import ImbraceClient
    client = ImbraceClient()
    me = client.platform.get_me()
    ```

  </div>
