## CRM

### 聯絡人管理

```typescript
// 聯絡人列表
const { data: contacts } = await client.contacts.list({ limit: 50 })

// 取得單個聯絡人
const contact = await client.contacts.get("contact_id")

// 更新資訊
await client.contacts.update("contact_id", {
  name: "Alice B.",
  email: "alice@example.com",
  phone: "+84901234567",
})

// 活動歷史、評論、檔案
const activity = await client.contacts.getActivity("contact_id")
const comments  = await client.contacts.getComments("contact_id")
const files     = await client.contacts.getFiles("contact_id")
```

### CRM 看板 (Pipeline)

看板是儲存 CRM 資料的地方 — leads、deals、tasks 或任何結構化資料。

```typescript
// 看板列表
const { data: boards } = await client.boards.list()

// 取得看板的項目（例如：pipeline leads）
const { data: leads } = await client.boards.listItems("board_id", { limit: 100 })

// 建立新項目（例如：新 lead）
const lead = await client.boards.createItem("board_id", {
  fields: { name: "Acme Corp", status: "new", value: 50000 },
})

// 更新項目（例如：進入下一階段）
await client.boards.updateItem("board_id", lead.id, {
  fields: { status: "qualified" },
})

// 搜尋項目
const results = await client.boards.search("board_id", { query: "enterprise" })

// 匯出 CSV
const csv = await client.boards.exportCsv("board_id")
```

### 對話

管理與客戶的對話 — 查看、分配、更新狀態。

```typescript
// 搜尋對話
const { data: convs } = await client.conversations.search({
  businessUnitId: "bu_xxx",
  q: "support",
  limit: 20,
})

// 未處理的對話
const { data: open } = await client.conversations.getOutstanding({
  businessUnitId: "bu_xxx",
  limit: 50,
})

// 分配給團隊成員
await client.conversations.assignTeamMember({
  conversation_id: "conv_xxx",
  user_id: "user_xxx",
})

// 更新狀態
await client.conversations.updateStatus({
  conversation_id: "conv_xxx",
  status: "resolved",
})
```

### 訊息

透過各渠道發送和接收訊息（web、WhatsApp、Facebook、email 等）。

```typescript
// 頻道列表
const channels = await client.channel.list()

// 發送訊息
await client.messages.send("conversation_id", {
  parts: [{ type: "text", text: "您好，有什麼可以幫您？" }],
})

// 訊息列表
const msgs = await client.messages.list("conversation_id")

// 標記已讀
await client.channel.markRead("conversation_id")
```

## AI

### AI 服務 (aiv2)

`client.ai` 和 `client.chatAi` 都呼叫 **aiv2** 服務。使用 `client.ai` 進行 OpenAI 相容呼叫；使用 `client.chatAi` 進行聊天工作階段和文件處理。

```typescript
// ── client.ai — OpenAI 相容補全 ─────────────────────────────────────────────

const response = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "您是 CRM 助手。" },
    { role: "user",   content: "摘要這位客戶的筆記：..." },
  ],
})
console.log(response.choices[0].message.content)

for await (const chunk of client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "為這個 lead 草擬一封跟進郵件。" }],
})) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? "")
}

const embeddings = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["客戶投訴帳單問題", "帳單問題升級"],
})

// ── client.chatAi — 托管聊天工作階段 ──────────────────────────────────────

const models   = await client.chatAi.listModels()
const response2 = await client.chatAi.chat({
  model: "gpt-4o",
  messages: [{ role: "user", content: "退款政策是什麼？" }],
})
const chat    = await client.chatAi.createChat({ title: "Support Chat" })
const history = await client.chatAi.listChats()
await client.chatAi.deleteChat(chat.id)

// ── 文件理解 ──────────────────────────────────────────────────────────────

const result = await client.chatAi.processDocument({
  modelName: "gpt-4o",
  url: "https://example.com/invoice.pdf",
  organizationId: "org_xxx",  // 該 endpoint 必填
})
console.log(result.data)
// { invoice_number: "INV-001", total: 1200, vendor: "Acme Corp", ... }
```

### AI Agent

使用知識庫（embeddings）進行串流聊天，並管理終端使用者聊天工作階段。

```typescript
// 與 AI 助手進行串流聊天（SSE）
// id 是工作階段/對話 ID — 在多輪對話中傳入相同的 id 以保留歷史。
//   只有在開始新對話時才省略；SDK 若未傳入則自動生成。
// user_id 是可選的 — 未提供時自動透過 chat-client/auth/user 取得。
const sessionId = crypto.randomUUID() // 在整個對話中保存此 id

const response = await client.aiAgent.streamChat({
  id: sessionId,
  assistant_id: "assistant_id",
  organization_id: "org_xxx",
  messages: [{ role: "user", content: "本季度哪些交易已成交？" }],
})

// 提示建議
const suggestions = await client.aiAgent.getAgentPromptSuggestion("assistant_id")

// 知識庫
const files      = await client.aiAgent.listEmbeddingFiles()
const classified = await client.aiAgent.classifyFile({
  url: "https://example.com/product-catalog.pdf",
  mime_type: "application/pdf",
})

// 管理終端使用者聊天工作階段
const chat = await client.aiAgent.createClientChat({
  id: crypto.randomUUID(),
  assistantId: "assistant_id",
  organizationId: "org_xxx",
  userId: "user_id",               // 從 chat-client/auth/user 取得
  selectedVisibilityType: "private",
  message: {
    id: crypto.randomUUID(),
    role: "user",
    content: "您好",
    createdAt: new Date().toISOString(),
    parts: [{ type: "text", text: "您好" }],
  },
})

const messages = await client.aiAgent.listClientMessages(chat.id)
await client.aiAgent.deleteClientChat(chat.id)
```
