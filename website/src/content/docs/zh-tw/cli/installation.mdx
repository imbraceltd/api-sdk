## 安裝 CLI

### 公開 npm 套件

```bash
npm install -g @imbrace/cli
```

### 從原始碼安裝

複製倉儲並執行安裝腳本：

```bash
git clone https://github.com/imbraceltd/imbrace-cli.git
cd imbrace-cli
./install.sh
```

`install.sh` 會依序執行 `npm install`、`npm run build`、`npm link`，然後將 `imbrace` 符號連結至 `/opt/homebrew/bin`（Apple Silicon）或 `/usr/local/bin`（Intel）。

### 手動建置

```bash
cd cli
npm install
npm run build
npm link
```

## 認證

所有指令（除了 `login`）在未認證時會自動提示登入。

### 使用 API 金鑰登入

建議用於 CI/CD 和程式碼代理工具：

```bash
imbrace login --api-key api_xxx...
```

### 使用電子郵件 + 密碼登入

```bash
imbrace login --email user@example.com --password mypass
```

### 檢查目前登入狀態

```bash
imbrace whoami [--json]
```

### 登出

```bash
imbrace logout
```

<Aside type="tip">
  如果指令回傳 401，請重新執行 `imbrace login --api-key api_xxx...`。
</Aside>

## 憑證儲存

| 作業系統 | 路徑 |
|---|---|
| macOS | `~/Library/Preferences/imbrace-nodejs/config.json` |
| Linux | `~/.config/imbrace-nodejs/config.json` |
| Windows | `%APPDATA%\imbrace-nodejs\Config\config.json` |
