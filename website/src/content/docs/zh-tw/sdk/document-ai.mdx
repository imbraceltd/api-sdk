import { Tabs, TabItem } from "@astrojs/starlight/components";

Document AI 使用視覺模型（例如 GPT-4o）從 PDF 與圖片中提取結構化資料 — 發票、表單、收據、合約或任何包含可見文字的文件。

SDK 提供兩個命名空間：
- **`client.chatAi` / `client.chat_ai`** — 直接文件處理與模型列表
- **`client.documentAi` / `client.document_ai`** — 基於 Agent 的文件處理，包含架構管理、Agent CRUD 與端到端編排

請先初始化客戶端（參閱[安裝](/sdk/installation/)或[快速入門](/sdk/quick-start/)）。

#### PDF 表單填寫範例

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const result = await client.chatAi.processDocument({
  modelName: "gpt-4o",
  url: "https://example.com/blank-invoice.pdf",
  organizationId: "org_xxx",
  fileUrlToFill: "https://example.com/blank-invoice.pdf",
  language: "en",
});

if (result.success && result.data.filledPdfUrl) {
  console.log("已填寫的 PDF：", result.data.filledPdfUrl);
}
```
</TabItem>
<TabItem label="Python">
```python
result = client.chat_ai.process_document(

    language="en",
)

if result["success"] and result["data"].get("filledPdfUrl"):
    print("已填寫的 PDF：", result["data"]["filledPdfUrl"])
```

## `client.documentAi` — Create Full（編排器）

端到端 Document AI Agent 建立（iMBRACE Web 應用程式的做法）：建立包含提取架構的看板，然後建立連結至該看板的 UseCase + AI Agent。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const result = await client.documentAi.createFull({
  name: "Invoice Extractor",
  instructions: "提取發票欄位。日期格式為 YYYY-MM-DD。",
  schemaFields: [
    { name: "invoice_number", type: "ShortText", description: "發票 ID" },
    { name: "total",          type: "Number",    description: "總金額" },
    { name: "date",           type: "Date",      description: "發票日期" },
  ],
  modelId: "gpt-4o",
  providerId: "system",
});

console.log(result.board_id);     // "brd_xxx"
console.log(result.ai_agent_id);  // 所建立 AI Agent 的 UUID
console.log(result.usecase_id);   // 所建立 UseCase 的 UUID
```
</TabItem>
<TabItem label="Python">
```python
result = client.document_ai.create_full(
    name="Invoice Extractor",
    instructions="提取發票欄位。日期格式為 YYYY-MM-DD。",
    schema_fields=[
        {"name": "invoice_number", "type": "ShortText", "description": "發票 ID"},
        {"name": "total",          "type": "Number",    "description": "總金額"},
        {"name": "date",           "type": "Date",      "description": "發票日期"},
    ],
    model_id="gpt-4o",
    provider_id="system",
)

print(result["board_id"])      # "brd_xxx"
print(result["ai_agent_id"])   # 所建立 AI Agent 的 UUID
```
</TabItem>
</Tabs>

## 非同步用法 (Python)

<Tabs syncKey="lang">
<TabItem label="AsyncImbraceClient">
```python
from imbrace import AsyncImbraceClient

async with AsyncImbraceClient() as client:
    # 直接處理 (chat_ai)
    models = await client.chat_ai.list_document_models()
    result = await client.chat_ai.process_document(
        model_name="gpt-4o",
        url="https://example.com/invoice.pdf",
        organization_id="org_xxx",
    )

    # 基於 Agent 的處理 (document_ai)
    agents = await client.document_ai.list_agents(document_ai_only=True)
    result2 = await client.document_ai.process(
        agent_id=agents[0]["_id"],
        url="https://example.com/receipt.pdf",
        organization_id="org_xxx",
    )
```
