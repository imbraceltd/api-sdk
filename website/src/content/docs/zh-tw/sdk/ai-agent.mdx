## AI Agent — `chatAi` / `chat_ai`

管理 AI Agent（CRUD）、執行 OpenAI 相容的 completions，以及處理 document/file。該命名空間也涵蓋 Knowledge Hub 資料夾和 knowledge bases。

### AI Agent CRUD

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
// 列出帳戶中的所有助手
const assistants = await client.chatAi.listAssistants();
// 每個助手有 `id`（UUID）和 `_id`（MongoDB ObjectId）。後續呼叫使用 `id` 欄位。

// 取得單一助手
const assistant = await client.chatAi.getAssistant("9f77692f-33d0-436a-8138-2efb268838e6");

const created = await client.chatAi.createAssistant({
  name: "Support Bot",

  provider_id: "system",
  model_id: "gpt-4o",
  description: "處理一級支援問題",

// 更新 — 更新時 workflow_name 也是必填

await client.chatAi.updateAssistantInstructions(
```

## OpenAI 相容 AI 服務 — `client.ai`

原始 OpenAI 風格的 completions、streaming 和 embeddings，針對 `aiv2` 服務。TypeScript 和 Python 均可使用。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
// 單次 completion
const response = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "您是 CRM 助手。" },
    { role: "user",   content: "摘要這位客戶的備註：..." },
  ],
  temperature: 0.7,
});
console.log(response.choices[0].message.content);

// 串流傳輸
const stream = client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "為這個 lead 起草跟進郵件。" }],
});
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0].delta.content ?? "");
}

// Embeddings
const result = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["客戶投訴計費問題", "計費問題升級"],
});
```
</TabItem>
<TabItem label="Python">
```python
# 單次 completion
response = client.ai.complete(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "您是 CRM 助手。"},
        {"role": "user",   "content": "摘要這位客戶的備註：..."},
    ],
    temperature=0.7,
)
print(response["choices"][0]["message"]["content"])

# 串流傳輸
for chunk in client.ai.stream(
    model="gpt-4o",
    messages=[{"role": "user", "content": "為這個 lead 起草跟進郵件。"}],
):
    print(chunk["choices"][0]["delta"].get("content", ""), end="", flush=True)

# Embeddings
result = client.ai.embed(
    model="text-embedding-ada-002",
    input=["客戶投訴計費問題", "計費問題升級"],
)
```

## Chat v2 — 串流傳輸（SSE）

回傳原始 response。將 body 作為 Server-Sent Events 串流消費。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const response = await client.aiAgent.streamChat({
  id: "chat_id",
  assistant_id: "asst_abc",
  organization_id: "org_abc",
  messages: [{ role: "user", content: "您能做什麼？" }],
});

const reader = response.body!.getReader();
const decoder = new TextDecoder();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  process.stdout.write(decoder.decode(value));
}
```
</TabItem>
<TabItem label="Python">
```python
response = client.ai_agent.stream_chat({
    "id": "chat_id",
    "assistant_id": "asst_abc",
    "organization_id": "org_abc",
    "messages": [{"role": "user", "content": "您能做什麼？"}],
})

for line in response.iter_lines():
    if line:
        print(line)
```

**非同步：**

```python
import asyncio
from imbrace import AsyncImbraceClient

async def main():
    async with AsyncImbraceClient() as client:
        response = await client.ai_agent.stream_chat({
            "id": "chat_id",
            "assistant_id": "asst_abc",
            "organization_id": "org_abc",
            "messages": [{"role": "user", "content": "您好"}],
        })
        async for line in response.aiter_lines():
            if line:
                print(line)

asyncio.run(main())
```
</TabItem>
</Tabs>
