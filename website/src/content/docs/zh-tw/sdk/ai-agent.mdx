## AI Agent — `chatAi` / `chat_ai`

管理 AI Agent（CRUD）、執行相容 OpenAI 的完成，以及處理文件/檔案處理。同一個命名空間也涵蓋知識庫資料夾與知識庫。

### AI Agent CRUD

## 相容 OpenAI 的 AI 服務 — `client.ai`

OpenAI 風格的完成、串流與嵌入向量。也提供 LLM 模型、提供者、護欄、RAG 檔案與 AI Agent 的管理 API。TypeScript 與 Python 皆可使用。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
// 單次完成
const response = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "你是一個樂於助人的 CRM 助理。" },
    { role: "user",   content: "摘要這段客戶備註：..." },
  ],
  temperature: 0.7,
});
console.log(response.choices[0].message.content);

// 串流
const stream = client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "為此潛在客戶草擬一封後續電子郵件。" }],
});
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0].delta.content ?? "");
}

// 嵌入向量
const result = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["客戶抱怨帳單", "帳單問題已升級"],
});
```
</TabItem>
<TabItem label="Python">
```python
from imbrace.types.ai import CompletionInput, CompletionMessage, EmbeddingInput

# 單次完成
response = client.ai.complete(CompletionInput(
    model="gpt-4o",
    messages=[
        CompletionMessage(role="system", content="你是一個樂於助人的 CRM 助理。"),
        CompletionMessage(role="user", content="摘要這段客戶備註：..."),
    ],
    temperature=0.7,
))
print(response.choices[0].message.content)

# 串流
for chunk in client.ai.stream(CompletionInput(
    model="gpt-4o",
    messages=[CompletionMessage(role="user", content="為此潛在客戶草擬一封後續電子郵件。")],
)):
    print(chunk.choices[0].delta.content or "", end="", flush=True)

# 嵌入向量
result = client.ai.embed(EmbeddingInput(
    model="text-embedding-ada-002",
    input=["客戶抱怨帳單", "帳單問題已升級"],
))
```
</TabItem>
</Tabs>

### 提供者

管理自訂 LLM 提供者。`listProviders` 包含一個合成的 `"system"` 項目（`provider_id: "system"`），代表來自 `getLlmModels` 的內建模型。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
// 列出所有提供者（包含系統預設作為第一個項目）
const providers = await client.ai.listProviders();
// 僅自訂提供者
const custom = await client.ai.listProviders({ includeSystem: false });

// 新增自訂提供者
const provider = await client.ai.createProvider({
  name: "My Azure OpenAI",
  type: "azure",
  api_key: "sk-...",
  base_url: "https://my-instance.openai.azure.com/",
});

// 更新、重新整理模型、刪除
await client.ai.updateProvider(provider._id, { name: "Azure OpenAI (prod)" });
await client.ai.refreshProviderModels(provider._id);
await client.ai.deleteProvider(provider._id);
```
</TabItem>

## Chat v2 — 串流 (SSE)

回傳原始回應。將主體作為伺服器推送事件串流使用。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const response = await client.aiAgent.streamChat({
  id: "chat_id",
  assistant_id: "asst_abc",
  messages: [{ role: "user", content: "你可以做什麼？" }],
});

const reader = response.body!.getReader();
const decoder = new TextDecoder();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
```
