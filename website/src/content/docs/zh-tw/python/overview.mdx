`imbrace` 套件是 Imbrace Gateway 的官方 Python 客戶端。提供**同步**和**非同步**客戶端，適用於傳統 Django/Flask 和 FastAPI 等現代非同步框架。

### 主要特性

| 特性                    | 詳情                                                                                        |
| ----------------------- | ------------------------------------------------------------------------------------------- |
| **雙客戶端**            | `ImbraceClient`（同步，`httpx.Client`）和 `AsyncImbraceClient`（非同步，`httpx.AsyncClient`）|
| **完整型別提示**        | 所有請求/回應透過 Pydantic models 和 type hints 進行型別標注                                |
| **雙重認證**            | API Key（伺服器端對伺服器端）或 Access Token（JWT / 傳統 token）                            |
| **自動重試**            | 429 和 5xx 最多重試 3 次，使用指數退避                                                      |
| **分頁輔助**            | `iterate_paged()` 遍歷所有頁面逐個產出項目                                                  |
| **上下文管理器**        | `with`/`async with` 語法自動清理連接                                                        |
| **多環境支援**          | 預設 `develop` / `sandbox` / `stable`                                                       |

### 工作原理

```
Your Code
   │
   ▼
ImbraceClient / AsyncImbraceClient     ← 唯一入口點
   ├── TokenManager                    ← 執行緒安全的 token 儲存
   └── HttpTransport / AsyncHttpTransport
         ├── 選擇認證策略               ← API Key / JWT Bearer / x-access-token
         ├── 指數退避重試               ← 429 和 5xx
         └── 連接池（httpx）            ← keepalive, max_connections=10
               │
               ▼
         Imbrace Gateway               ← app-gatewayv2.imbrace.co
```

### 內部架構

| 層級             | 檔案                           | 職責                                              |
| ---------------- | ------------------------------ | ------------------------------------------------- |
| 入口點           | `client.py`, `async_client.py` | 初始化 resources，登入輔助方法                    |
| HTTP 傳輸        | `http.py`                      | 認證標頭、重試、逾時、錯誤映射                    |
| Token 管理器     | `auth/token_manager.py`        | 執行緒安全的 token 讀寫                           |
| 服務登錄表       | `service_registry.py`          | 從環境預設解析每個服務的 URL                      |
| Resources        | `resources/*.py`               | 每個領域的 API 方法封裝（31 個 resources）        |
| 型別             | `types/*.py`                   | Pydantic models 和 dataclasses                    |
| 例外             | `exceptions.py`                | `AuthError`、`ApiError`、`NetworkError`            |

### 同步 vs 非同步 — 何時使用哪個

| 客戶端               | 框架                           | 匯入                                             |
| -------------------- | ------------------------------ | ------------------------------------------------ |
| `ImbraceClient`      | Django、Flask、腳本、Celery    | `from imbrace import ImbraceClient`              |
| `AsyncImbraceClient` | FastAPI、asyncio、aiohttp      | `from imbrace import AsyncImbraceClient`         |

:::tip
兩個客戶端具有完全相同的 resource 介面 — 唯一的差異是方法呼叫前的 `await`。您可以在不更改業務邏輯的情況下相互替換。
:::
