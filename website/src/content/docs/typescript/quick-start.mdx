title: TypeScript SDK — Quick Start
description: Make your first Imbrace API call in under a minute.
---

### 1. Initialize the Client

```typescript
import { ImbraceClient } from "@imbrace/sdk"

const client = new ImbraceClient({
  accessToken: "your-access-token",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})
```

---

### 2. Fetch Your Boards

Boards are CRM pipelines — leads, deals, tasks, or any structured data. This call requires only your access token.

```typescript
const { data: boards } = await client.boards.list()

for (const board of boards) {
  console.log(board._id, board.doc_name)
}
```

That's it. If you see your boards listed, the SDK is connected and authenticated.

---

### 3. Next Steps

Once connected, you can explore the full feature set:

- **AI Agent** — stream live responses from an AI assistant trained on your knowledge base → requires an `assistant_id` from your Imbrace dashboard
- **Workflows** — trigger and manage Activepieces automations
- **Boards & Items** — full CRM pipeline management
- **Knowledge Hub** — manage files and knowledge bases for AI
