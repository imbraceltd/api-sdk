---
title: TypeScript SDK — Resources
description: Complete reference for all domain resource namespaces available on ImbraceClient.
---

Every resource is accessed as a property on the `ImbraceClient` instance. All methods return Promises and are fully typed.

```typescript
import { ImbraceClient } from "@imbrace/sdk";
const client = new ImbraceClient();
```

---

### Marketplace

Manage products and orders in the Imbrace Marketplace.

```typescript
// Products
const { data: products } = await client.marketplace.listProducts({
  category: "electronics",
  page: 1,
  limit: 20,
});
const product = await client.marketplace.getProduct("prod_id");
await client.marketplace.createProduct({ name: "Product A", price: 99 });
await client.marketplace.updateProduct("prod_id", { price: 89 });
await client.marketplace.deleteProduct("prod_id");

// Orders
const { data: orders } = await client.marketplace.listOrders({
  status: "pending",
});
const order = await client.marketplace.getOrder("order_id");
await client.marketplace.createOrder({
  items: [{ productId: "prod_id", quantity: 2 }],
  shippingAddress: { city: "Ho Chi Minh", country: "VN" },
});
await client.marketplace.updateOrderStatus("order_id", "confirmed");
```

```typescript

await client.platform.revokePermission("user_id", "perm_id");
```

### Agent

Create and run AI agent templates.

```typescript
// Manage agents
const agents = await client.agent.listAgents();

  name: "Data Analyzer",
```

### AI

LLM completions, streaming output, and text embeddings.

```typescript
// Non-streaming completion
const completion = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "You are a helpful assistant." },
    { role: "user", content: "Summarize this report." },
  ],
  temperature: 0.7,
  maxTokens: 1000,
});
console.log(completion.choices[0].message.content);

// Streaming completion — yields chunks via AsyncGenerator
for await (const chunk of client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "Count from 1 to 5." }],
})) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? "");
}

// Text embeddings
const embeddings = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["Hello world", "Imbrace SDK"],
});
```

### Auth

OTP and password-based login. Token is stored automatically after successful login.

```typescript
// Request OTP
await client.auth.requestOtp({ email: "user@example.com" });

// Verify OTP and sign in
const result = await client.auth.verifyOtp({
  email: "user@example.com",
  otp: "123456",
});

// Sign out
await client.auth.signOut();
```

> Shorthand login helpers are also available directly on the client:
> `client.requestOtp(email)`, `client.loginWithOtp(email, otp)`, `client.login(email, password)`

### AI Agent

Chat, embedding files, parquet data, and distributed traces via the ai-agent service.

```typescript
// System info
const config = await client.aiAgent.getConfig()
const health = await client.aiAgent.getHealth()

// Chat v1 — query history
const { chats } = await client.aiAgent.listChats({ organization_id: "org_xxx" })
const chat = await client.aiAgent.getChat(chats[0].id)

// Chat v2 — streaming (SSE)
const response = await client.aiAgent.streamChat({
  id: "chat_id",
  assistant_id: "assistant_id",
  organization_id: "org_xxx",
  messages: [{ role: "user", content: "Hello" }],
})
// consume as a stream: response.body (ReadableStream)

// Prompt suggestions
const suggestions = await client.aiAgent.getAgentPromptSuggestion("assistant_id")

// Embedding files
const files = await client.aiAgent.listEmbeddingFiles()
const classified = await client.aiAgent.classifyFile({ url: "https://…/doc.pdf", mime_type: "application/pdf" })

// Parquet
const result = await client.aiAgent.generateParquet({
  fileName: "export_2026",
  data: [{ id: 1, value: 42 }],
})
await client.aiAgent.deleteParquetFile(result.fileName)

// Chat client (end-user chats)
const newChat = await client.aiAgent.createClientChat({
  id: crypto.randomUUID(),
  message: { id: crypto.randomUUID(), role: "user", parts: [{ type: "text", text: "Hi" }] },
  assistantId: "assistant_id",
  organizationId: "org_xxx",
})
const messages = await client.aiAgent.listClientMessages(newChat.id)
await client.aiAgent.deleteClientChat(newChat.id)
```

### Pagination

List endpoints return a consistent shape:

```typescript
const { data, pagination } = await client.marketplace.listProducts({
  page: 1,
  limit: 20,
});

console.log(data); // Product[]
console.log(pagination.hasNext); // boolean
console.log(pagination.total); // number
```
