## AI Agent (`client.aiAgent`)

### Stream Chat (SSE)

> For SSE handling and a session-id walkthrough, see [Full Flow Guide §1](/typescript/full-flow-guide/#1-create-an-ai-assistant-and-start-chatting). The reference below shows the call signature.

```typescript
// Keep sessionId across turns to maintain conversation history.
// Omit id on the first message — the SDK generates one automatically.
// user_id is also optional — resolved automatically from auth context.
const sessionId = crypto.randomUUID()

const response = await client.aiAgent.streamChat({
  id: sessionId,
  assistant_id: "asst_xxx",
  organization_id: "org_xxx",
  messages: [{ role: "user", content: "What deals closed this quarter?" }],
})

const reader = response.body!.getReader()
const decoder = new TextDecoder()

while (true) {
  const { done, value } = await reader.read()
  if (done) break
  const text = decoder.decode(value)
  for (const line of text.split("\n")) {
    if (!line.startsWith("data: ")) continue
    const raw = line.slice(6).trim()
    if (raw === "[DONE]") break
    try {
      const event = JSON.parse(raw)
      if (event.type === "text-delta") process.stdout.write(event.textDelta)
    } catch {}
  }
}
```

### End-User Chat Client

```typescript
// Create a new end-user chat session
const chat = await client.aiAgent.createClientChat({
  id: crypto.randomUUID(),
  assistantId: "asst_xxx",
  organizationId: "org_xxx",
  userId: "user_xxx",
  selectedVisibilityType: "private",
  message: {
    id: crypto.randomUUID(),
    role: "user",
    content: "Hello, I need help with my order.",
    createdAt: new Date().toISOString(),
    parts: [{ type: "text", text: "Hello, I need help with my order." }],
  },

// List all end-user chats
const clientChats = await client.aiAgent.listClientChats()

const clientChat = await client.aiAgent.getClientChat(chat.id)

// List messages in an end-user chat
const messages = await client.aiAgent.listClientMessages(chat.id)

// Delete an end-user chat
await client.aiAgent.deleteClientChat(chat.id)
```

## Assistant — AI Service v2 (`client.chatAi`)

### Assistant CRUD

```typescript
// List all assistants in your account
const assistants = await client.chatAi.listAssistants()
// Each assistant has an `id` (UUID) and `_id` (MongoDB ObjectId)
// Use the `id` field for all subsequent calls

// Get a single assistant
const assistant = await client.chatAi.getAssistant("9f77692f-33d0-436a-8138-2efb268838e6")

// Create an assistant — workflow_name must be unique within your org
const created = await client.chatAi.createAssistant({
  name: "Support Bot",
  workflow_name: "support_bot_v1",
  description: "Handles tier-1 support queries",
})

// Update an assistant — workflow_name is required on update too
const updated = await client.chatAi.updateAssistant(created.id, {
  name: "Support Bot v2",
  workflow_name: "support_bot_v1",
})

// Update only the system instructions
await client.chatAi.updateAssistantInstructions(created.id, "You are a helpful support agent.")

// Delete an assistant
await client.chatAi.deleteAssistant(created.id)
```
