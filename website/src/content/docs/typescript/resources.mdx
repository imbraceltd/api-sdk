## CRM

### Contacts

List, retrieve, and update contacts.

```typescript
// List contacts
const { data: contacts } = await client.contacts.list({ limit: 50 })

// Get a specific contact
const contact = await client.contacts.get("contact_id")

// Update contact info
await client.contacts.update("contact_id", {
  name: "Alice B.",
  email: "alice@example.com",
  phone: "+84901234567",
})

// Get contact activity (comments, files, conversations)
const activity = await client.contacts.getActivity("contact_id")
const comments  = await client.contacts.getComments("contact_id")
const files     = await client.contacts.getFiles("contact_id")
```

### CRM Boards

Boards are the core data store for CRM pipelines — leads, deals, tasks, or any structured data.

```typescript
// List all boards
const { data: boards } = await client.boards.list()

// Get items from a board (e.g., a leads pipeline)
const { data: leads } = await client.boards.listItems("board_id", { limit: 100 })

// Create a new item (e.g., a new lead)
const lead = await client.boards.createItem("board_id", {
  fields: {
    name: "Acme Corp",
    status: "new",
    value: 50000,
  },
})

// Update an item (e.g., move to next stage)
await client.boards.updateItem("board_id", lead.id, {
  fields: { status: "qualified" },
})

// Search items
const results = await client.boards.search("board_id", { query: "enterprise" })

// Export board to CSV
const csv = await client.boards.exportCsv("board_id")
```

### Conversations

Manage customer conversations — view, assign, and update status.

```typescript
// Search conversations
const { data: convs } = await client.conversations.search({
  businessUnitId: "bu_xxx",
  q: "support",
  limit: 20,
})

// Get outstanding (unresolved) conversations
const { data: open } = await client.conversations.getOutstanding({
  businessUnitId: "bu_xxx",
  limit: 50,
})

// Assign a conversation to a team member
await client.conversations.assignTeamMember({
  conversation_id: "conv_xxx",
  user_id: "user_xxx",
})

// Update conversation status
await client.conversations.updateStatus({
  conversation_id: "conv_xxx",
  status: "resolved",
})
```

### Messaging

Send and receive messages across channels (web, WhatsApp, Facebook, email, etc.).

```typescript
// List channels
const channels = await client.channel.list()

// Send a message in a conversation
await client.messages.send("conversation_id", {
  parts: [{ type: "text", text: "Hello, how can I help you today?" }],
})

// List messages in a conversation
const msgs = await client.messages.list("conversation_id")

// Mark conversation as read
await client.channel.markRead("conversation_id")
```

### Workflows & Automation

Trigger channel automations to handle inbound messages, route conversations, and fire follow-up actions.

```typescript
// List all channel automations
const { data: automations } = await client.workflows.listChannelAutomation()

// Filter by channel type
const whatsappFlows = await client.workflows.listChannelAutomation({
  channelType: "whatsapp",
})
```

## AI

### AI Service (aiv2)

`client.ai` and `client.chatAi` both route to the **aiv2** service. Use `client.ai` for raw OpenAI-compatible calls; use `client.chatAi` for managed chat sessions and document processing.

```typescript
// ── client.ai — OpenAI-compatible completions ──────────────────────────────

// Single completion
const response = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "You are a helpful CRM assistant." },
    { role: "user",   content: "Summarize this customer note: ..." },
  ],
  temperature: 0.7,
})
console.log(response.choices[0].message.content)

// Streaming
for await (const chunk of client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "Draft a follow-up email for this lead." }],
})) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? "")
}

// Embeddings (semantic search, similarity)
const embeddings = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["customer complained about billing", "billing issue escalated"],
})

// ── client.chatAi — managed chat sessions ─────────────────────────────────

// List available models
const models = await client.chatAi.listModels()

// Chat with history
const response2 = await client.chatAi.chat({
  model: "gpt-4o",
  messages: [{ role: "user", content: "What is our refund policy?" }],
})

// Persistent sessions
const chat    = await client.chatAi.createChat({ title: "Support Chat" })
const history = await client.chatAi.listChats()
await client.chatAi.deleteChat(chat.id)

// ── Document Understanding ─────────────────────────────────────────────────

// Extract structured data from PDFs, invoices, images
const result = await client.chatAi.processDocument({
  modelName: "gpt-4o",
  url: "https://example.com/invoice.pdf",
  organizationId: "org_xxx",  // required for this endpoint only
})
console.log(result.data)
// { invoice_number: "INV-001", total: 1200, vendor: "Acme Corp", ... }
```

### AI Agent

Streaming chat assistant with a knowledge base (embeddings), parquet data, and end-user chat management.

```typescript
// Stream a chat with an AI assistant (SSE)
const response = await client.aiAgent.streamChat({
  id: crypto.randomUUID(),
  assistant_id: "assistant_id",
  organization_id: "org_xxx",
  messages: [{ role: "user", content: "What deals closed this quarter?" }],
})
// consume response.body as a ReadableStream / SSE

// Prompt suggestions for an assistant
const suggestions = await client.aiAgent.getAgentPromptSuggestion("assistant_id")

// Knowledge base — list and classify files
const files      = await client.aiAgent.listEmbeddingFiles()
const classified = await client.aiAgent.classifyFile({
  url: "https://example.com/product-catalog.pdf",
  mime_type: "application/pdf",
})

// Manage end-user chat sessions
const chat = await client.aiAgent.createClientChat({
  id: crypto.randomUUID(),
  message: {
    id: crypto.randomUUID(),
    role: "user",
    parts: [{ type: "text", text: "Hello" }],
  },
  assistantId: "assistant_id",
})

const messages = await client.aiAgent.listClientMessages(chat.id)
await client.aiAgent.deleteClientChat(chat.id)
```
