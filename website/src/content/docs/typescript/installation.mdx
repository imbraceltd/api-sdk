### Install

```bash
npm install @imbrace/sdk
# or
yarn add @imbrace/sdk
# or
bun add @imbrace/sdk
```

---

### Initialize the Client

```typescript
import { ImbraceClient } from "@imbrace/sdk"

// Option 1 — access token (Build on Imbrace: Imbrace handles auth, db, logic)
const client = new ImbraceClient({
  accessToken: "acc_your_token",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})

// Option 2 — API key (Wrap Imbrace: your backend, your users, Imbrace as a feature)
const client = new ImbraceClient({
  apiKey: "api_xxx...",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})
```

The client is stateful — create it once and reuse it across your application.

For when to use each, see [Authentication](/typescript/authentication/). For step-by-step credential setup (env vars, dotenv), see [Setup Guide](/getting-started/setup/#configure-credentials).
