## CRM

### Contacts

List, retrieve, and update contacts.

```python
# List contacts
result   = client.contacts.list(limit=50)
contacts = result.get("data", [])

# Get a specific contact
contact = client.contacts.get("contact_id")

# Update contact info
client.contacts.update("contact_id", {
    "name":  "Alice B.",
    "email": "alice@example.com",
    "phone": "+84901234567",
})

# Get contact activity (comments, files, conversations)
activity  = client.contacts.get_activities("conversation_id")
comments  = client.contacts.get_comments("contact_id")
files     = client.contacts.get_files("contact_id")
```

---

### CRM Boards

Boards are the core data store for CRM pipelines — leads, deals, tasks, or any structured data.

```python
# List all boards
boards = client.boards.list().get("data", [])

# Get items from a board (e.g., a leads pipeline)
leads = client.boards.list_items("board_id", limit=100).get("data", [])

# Create a new item (e.g., a new lead)
lead = client.boards.create_item("board_id", {
    "fields": {
        "name":   "Acme Corp",
        "status": "new",
        "value":  50000,
    }
})

# Update an item (e.g., move to next stage)
client.boards.update_item("board_id", lead["id"], {
    "fields": {"status": "qualified"}
})

# Search items
results = client.boards.search("board_id", {"query": "enterprise"})

# Export board to CSV
csv = client.boards.export_csv("board_id")
```

### Conversations

Manage customer conversations — view, assign, and update status.

```python
# Search conversations
result = client.conversations.search(
    business_unit_id="bu_xxx",
    q="support",
    limit=20,
)
convs = result.get("data", [])

# Get outstanding (unresolved) conversations
open_convs = client.conversations.get_outstanding(
    business_unit_id="bu_xxx", limit=50
).get("data", [])

# Update conversation status
client.conversations.update_status({
    "conversation_id": "conv_xxx",
    "status": "resolved",
})
```

## AI

### AI Service (aiv2)

`client.ai` and `client.chat_ai` both route to the **aiv2** service. Use `client.ai` for raw OpenAI-compatible calls; use `client.chat_ai` for managed chat sessions and document processing.

```python
# ── client.ai — OpenAI-compatible completions ──────────────────────────────

# Single completion
response = client.ai.complete(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful CRM assistant."},
        {"role": "user",   "content": "Summarize this customer note: ..."},
    ],
    temperature=0.7,
)
print(response["choices"][0]["message"]["content"])

# Streaming
for chunk in client.ai.stream(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Draft a follow-up email for this lead."}],
):
    print(chunk["choices"][0]["delta"].get("content", ""), end="", flush=True)

# Embeddings (semantic search, similarity)
result = client.ai.embed(
    model="text-embedding-ada-002",
    input=["customer complained about billing", "billing issue escalated"],
)

# ── client.chat_ai — managed chat sessions ────────────────────────────────

# List available models
models = client.chat_ai.list_models()

# Chat with history
response2 = client.chat_ai.chat({
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "What is our refund policy?"}],
})

# Persistent sessions
chat    = client.chat_ai.create_chat({"title": "Support Chat"})
history = client.chat_ai.list_chats()
client.chat_ai.delete_chat(chat["id"])

# ── Document Understanding ─────────────────────────────────────────────────

# Extract structured data from PDFs, invoices, images
result = client.chat_ai.process_document(
    model_name="gpt-4o",
    url="https://example.com/invoice.pdf",
    organization_id="org_xxx",  # required for this endpoint only
)
print(result["data"])
# {"invoice_number": "INV-001", "total": 1200, "vendor": "Acme Corp", ...}
```

### AI Agent

Streaming chat assistant with a knowledge base (embeddings), parquet data, and end-user chat management.

```python
import uuid

# Stream a chat with an AI assistant (returns raw httpx.Response for SSE)
# id is the session/conversation ID — pass the same id across turns to maintain history.
#   Omit it only for a brand-new conversation; the SDK will generate one automatically.
# user_id is optional — resolved automatically via chat-client/auth/user when not provided.
session_id = str(uuid.uuid4())  # persist this for the lifetime of the conversation

resp = client.ai_agent.stream_chat({
    "id": session_id,
    "assistant_id": "assistant_id",
    "organization_id": "org_xxx",
    "messages": [{"role": "user", "content": "What deals closed this quarter?"}],
})
for line in resp.iter_lines():
    print(line)

# Prompt suggestions for an assistant
suggestions = client.ai_agent.get_agent_prompt_suggestion("assistant_id")

# Knowledge base — list and classify files
files      = client.ai_agent.list_embedding_files()
classified = client.ai_agent.classify_file(
    url="https://example.com/product-catalog.pdf",
    mime_type="application/pdf",
)

# Manage end-user chat sessions
from datetime import datetime, timezone
chat = client.ai_agent.create_client_chat({
    "id": str(uuid.uuid4()),
    "assistantId": "assistant_id",
    "organizationId": "org_xxx",
    "userId": "user_id",               # from chat-client/auth/user
    "selectedVisibilityType": "private",
    "message": {
        "id":        str(uuid.uuid4()),
        "role":      "user",
        "content":   "Hello",
        "createdAt": datetime.now(timezone.utc).isoformat(),
        "parts":     [{"type": "text", "text": "Hello"}],
    },
})

messages = client.ai_agent.list_client_messages(chat["id"])
client.ai_agent.delete_client_chat(chat["id"])
```
