Imbrace là nền tảng tương tác khách hàng — nó xử lý các kênh chat, CRM pipeline, AI agent, quy trình tự động hóa và xử lý tài liệu tại một nơi. SDK cho phép bạn nhúng các tính năng đó trực tiếp vào ứng dụng của mình mà không cần tự xây dựng hạ tầng.

### Bạn Có Thể Xây Dựng

- **AI Agents** — stream phản hồi trực tiếp từ AI agent được huấn luyện trên kiến thức của bạn
- **Quy trình tự động** — kích hoạt và quản lý các quy trình tự động hóa nhiều bước qua workflow engine
- **CRM pipelines** — tạo board, quản lý lead, tìm kiếm và xuất dữ liệu deal
- **Xử lý tài liệu** — trích xuất dữ liệu có cấu trúc từ PDF và hóa đơn, tạo embeddings

### CLI Tool

Imbrace cũng cung cấp giao diện dòng lệnh (`@imbrace/cli`) để tương tác với nền tảng từ terminal. Nó được thiết kế cho nhà phát triển và coding agent — cài đặt cùng SDK để tạo mẫu nhanh, gỡ lỗi và tự động hóa.

Xem [CLI Overview](/cli/overview/) để biết chi tiết.

### SDK Khả Dụng

| SDK | Package | Runtime |
|---|---|---|
| TypeScript / JavaScript | `@imbrace/sdk` | Node.js 18+, trình duyệt |
| Python | `imbrace` | Python 3.9+ |

### Xác Thực

Có hai cách xác thực:

- **Access token** — cho ứng dụng có người dùng. Lấy được sau khi người dùng đăng nhập qua OTP hoặc OAuth. Truyền dưới dạng `accessToken` khi khởi tạo client.
- **API key** — cho tích hợp server-to-server. Tạo từ bảng điều khiển Imbrace. Truyền dưới dạng `apiKey`.

### Gateway

Tất cả request SDK đều đi qua:

```
https://app-gatewayv2.imbrace.co
```

Gateway được đặt mặc định — bạn không cần cấu hình trừ khi nhắm đến môi trường khác.

### Ví Dụ Nhanh

```typescript
import { ImbraceClient } from "@imbrace/sdk"

const client = new ImbraceClient({
  accessToken: "your-access-token",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})

const stream = await client.aiAgent.streamChat({
  assistant_id: "asst_xxx",
  messages: [{ role: "user", content: "What deals closed this quarter?" }],
})
```
