---
title: TypeScript SDK — Xác Thực
description: API Key và Access Token trong Imbrace TypeScript SDK.
---

SDK hỗ trợ hai chiến lược xác thực. Truyền `apiKey` hoặc `accessToken` khi khởi tạo `ImbraceClient` — transport tự xử lý headers.

### Các Chiến Lược Xác Thực

| Chiến lược       | Khi nào dùng                              | Header gửi đi             |
| ---------------- | ----------------------------------------- | ------------------------- |
| **API Key**      | Backend scripts, cron jobs, server calls  | `x-api-key: sk-xxx...`    |
| **Access Token** | Sau khi user đăng nhập (OTP hoặc password)| `x-access-token: acc_...` |

---

### API Key (Server-Side)

```typescript
import { ImbraceClient } from "@imbrace/sdk";

const client = new ImbraceClient({
  apiKey: process.env.IMBRACE_API_KEY,
});
```

---

### Access Token (Client-Side)

Dùng sau khi user xác thực:

```typescript
const client = new ImbraceClient({
  accessToken: "acc_xxxxxxxxxxxxx",
});
```

---

### Luồng OTP Login

Cách được khuyến nghị cho user-facing apps — SDK tự lưu token sau khi đăng nhập thành công.

```typescript
import { ImbraceClient } from "@imbrace/sdk";

const client = new ImbraceClient();

// Bước 1: Gửi OTP đến email người dùng
await client.requestOtp("user@example.com");

// Bước 2: Người dùng nhập OTP — token được lưu tự động
await client.loginWithOtp("user@example.com", "123456");

// Bước 3: Tất cả các call tiếp theo đã được xác thực
const me = await client.platform.getMe();
```

Đăng nhập bằng email + password cũng được hỗ trợ:

```typescript
await client.login("user@example.com", "password123");
```

---

### Quản Lý Token

```typescript
// Thay token (ví dụ: sau khi OAuth refresh)
client.setAccessToken("new_token...");

// Xóa token (ví dụ: khi logout)
client.clearAccessToken();
```

---

### Thứ Tự Ưu Tiên Cấu Hình

1. **Constructor options tường minh** — `new ImbraceClient({ apiKey: "..." })`
2. **Environment variables** — `IMBRACE_API_KEY`, `IMBRACE_GATEWAY_URL`
3. **Defaults** — `env: "stable"`, gateway `https://app-gatewayv2.imbrace.co`
