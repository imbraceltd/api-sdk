### Yêu Cầu

- Node.js 18+
- `@imbrace/sdk` đã cài ([Cài Đặt →](installation/))
- `IMBRACE_API_KEY` đã set trong `.env`

---

### 1. Khởi Tạo Client

```typescript
import { ImbraceClient } from "@imbrace/sdk";
import "dotenv/config"; // tự đọc .env

const client = new ImbraceClient({
  apiKey: process.env.IMBRACE_API_KEY,
  env: "stable", // "develop" | "sandbox" | "stable"
  timeout: 30000, // ms, mặc định 30000
});
```

Client có trạng thái — tạo một lần và tái sử dụng trong toàn bộ ứng dụng.

---

### 2. Call API Đầu Tiên

```typescript
// Lấy thông tin tài khoản hiện tại
const me = await client.platform.getMe();
console.log("Đăng nhập với:", me.displayName);

// Liệt kê sản phẩm marketplace
const { data: products } = await client.marketplace.listProducts({
  page: 1,
  limit: 10,
});
console.log(`Tìm thấy ${products.length} sản phẩm`);
```

---

### 3. Health Check (Tùy Chọn)

Ping gateway khi khởi động để kiểm tra kết nối:

```typescript
await client.init();
console.log("Kết nối gateway OK");
```

---

### 4. Ví Dụ Script Hoàn Chỉnh

```typescript
// main.ts
import { ImbraceClient, AuthError, ApiError } from "@imbrace/sdk";
import "dotenv/config";

const client = new ImbraceClient();

async function main() {
  try {
    await client.init();

    const me = await client.platform.getMe();
    console.log("User:", me.displayName, me.email);

    const { data: channels } = await client.channel.listChannels({
      type: "group",
    });
    console.log(`Số kênh: ${channels.length}`);
  } catch (e) {
    if (e instanceof AuthError) {
      console.error("Xác thực thất bại — kiểm tra IMBRACE_API_KEY");
    } else if (e instanceof ApiError) {
      console.error(`Lỗi API [${e.statusCode}]:`, e.message);
    } else {
      throw e;
    }
  }
}

main();
```

```bash
# Chạy với ts-node
npx ts-node main.ts

# Hoặc build trước
npm run build && node dist/main.js
```

### 5. Chuyển Đổi Môi Trường

```typescript
// Môi trường development
const client = new ImbraceClient({ env: "develop" });

// Môi trường sandbox
const client = new ImbraceClient({ env: "sandbox" });

// Stable (production) — mặc định
const client = new ImbraceClient({ env: "stable" });

// URL gateway tùy chỉnh
const client = new ImbraceClient({
  gateway: "https://my-custom-gateway.example.com",
});
```

| Môi trường | Gateway URL                              |
| ---------- | ---------------------------------------- |
| `develop`  | `https://app-gateway.dev.imbrace.co`     |
| `sandbox`  | `https://app-gateway.sandbox.imbrace.co` |
| `stable`   | `https://app-gatewayv2.imbrace.co`       |

:::tip
Set `IMBRACE_GATEWAY_URL` trong `.env` để chuyển môi trường mà không cần thay code. Biến này ưu tiên hơn `env` preset.
:::
