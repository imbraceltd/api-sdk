### Cài Đặt

```bash
npm install @imbrace/sdk
# hoặc
yarn add @imbrace/sdk
# hoặc
pnpm add @imbrace/sdk
```

Để cấu hình thông tin xác thực, môi trường và override URL dịch vụ, xem [Hướng Dẫn Cài Đặt](/vi/getting-started/setup/).

---

### Build Scripts

```bash
npm run build      # compile TypeScript → dist/
npm run dev        # watch mode — tự rebuild khi thay đổi
npm run typecheck  # chỉ type-check (không emit)
npm run test       # chạy unit tests
npm run clean      # xóa dist/
```

---

### Biến Môi Trường

| Biến                  | Bắt Buộc | Mặc Định | Mô Tả                                            |
| --------------------- | -------- | -------- | ------------------------------------------------ |
| `IMBRACE_API_KEY`     | Có\*     | —        | Server-side API Key                              |
| `IMBRACE_ENV`         | Không    | `stable` | Preset: `develop`, `sandbox`, `stable`           |
| `IMBRACE_BASE_URL`    | Không    | auto     | Override URL gateway (ưu tiên hơn `IMBRACE_ENV`) |

\*Hoặc truyền `accessToken` trực tiếp cho client-side usage.
