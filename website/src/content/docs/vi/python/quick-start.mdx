### Yêu Cầu

- Python 3.10+
- `imbrace` đã cài ([Cài Đặt →](installation/))
- `IMBRACE_API_KEY` đã set trong `.env` hoặc environment

---

### 1. Khởi Tạo Client Đồng Bộ

```python
import os
from dotenv import load_dotenv
from imbrace import ImbraceClient

load_dotenv()  # đọc file .env

client = ImbraceClient(
    api_key=os.environ["IMBRACE_API_KEY"],
    env="stable",  # "develop" | "sandbox" | "stable"
    timeout=30,    # giây
)
```

---

### 2. Call API Đầu Tiên

```python
# Lấy thông tin người dùng hiện tại
me = client.platform.get_me()
print(f"Đăng nhập với: {me['displayName']}")

# Liệt kê sản phẩm marketplace
result = client.marketplace.list_products(page=1, limit=10)
print(f"Tìm thấy {len(result['data'])} sản phẩm")
```

### 3. Dùng Context Manager

Context manager đảm bảo connection pool được đóng đúng cách:

```python
with ImbraceClient() as client:
    me = client.platform.get_me()
    print(me)
# Kết nối tự đóng ở đây
```

### 4. Ví Dụ Script Hoàn Chỉnh

```python
# main.py
import os
from dotenv import load_dotenv
from imbrace import ImbraceClient, AuthError, ApiError, NetworkError

load_dotenv()

def main():
    with ImbraceClient() as client:
        try:
            me = client.platform.get_me()
            print(f"User: {me['displayName']} ({me['email']})")

            channels = client.channel.list_channels(type="group")
            print(f"Số kênh: {len(channels.get('data', []))}")

        except AuthError:
            print("Xác thực thất bại — kiểm tra IMBRACE_API_KEY")
        except ApiError as e:
            print(f"Lỗi API [{e.status_code}]: {e}")
        except NetworkError:
            print("Không thể kết nối tới Imbrace Gateway")

if __name__ == "__main__":
    main()
```

```bash
python main.py
```

### 5. Async Client (FastAPI / asyncio)

```python
import asyncio
from imbrace import AsyncImbraceClient

async def main():
    async with AsyncImbraceClient() as client:
        me = await client.platform.get_me()
        print(f"User: {me['displayName']}")

        products = await client.marketplace.list_products(limit=10)
        print(f"Sản phẩm: {len(products['data'])}")

asyncio.run(main())
```

### 6. Chuyển Đổi Môi Trường

```python
# Development gateway
client = ImbraceClient(env="develop")

# Sandbox gateway
client = ImbraceClient(env="sandbox")

# URL gateway tùy chỉnh
client = ImbraceClient(gateway="https://my-custom-gateway.example.com")
```

| Môi trường | Gateway URL                              |
| ---------- | ---------------------------------------- |
| `develop`  | `https://app-gateway.dev.imbrace.co`     |
| `sandbox`  | `https://app-gateway.sandbox.imbrace.co` |
| `stable`   | `https://app-gatewayv2.imbrace.co`       |

:::tip
Set `IMBRACE_GATEWAY_URL` trong `.env` để chuyển môi trường mà không cần thay code.
:::
