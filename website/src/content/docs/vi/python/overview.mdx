Package `imbrace` là client Python chính thức cho Imbrace Gateway. Cung cấp cả **synchronous** và **asynchronous** client, phù hợp cho Django/Flask truyền thống và các framework async hiện đại như FastAPI.

### Tính Năng Nổi Bật

| Tính năng               | Chi tiết                                                                                    |
| ----------------------- | ------------------------------------------------------------------------------------------- |
| **Dual Client**         | `ImbraceClient` (sync, `httpx.Client`) và `AsyncImbraceClient` (async, `httpx.AsyncClient`) |
| **Full Typing**         | Mọi request/response được type qua Pydantic models và type hints                            |
| **Dual Authentication** | API Key (server-to-server) hoặc Access Token (JWT / legacy token)                           |
| **Auto Retry**          | 429 và 5xx retry tối đa 3 lần với exponential backoff                                       |
| **Pagination Helper**   | `iterate_paged()` yield từng item qua tất cả các trang                                      |
| **Context Manager**     | Cú pháp `with`/`async with` tự động dọn dẹp kết nối                                         |
| **Multi-Environment**   | Preset `develop` / `sandbox` / `stable`                                                     |

### Cách Hoạt Động

```
Your Code
   │
   ▼
ImbraceClient / AsyncImbraceClient     ← entry point duy nhất
   ├── TokenManager                    ← lưu token thread-safe
   └── HttpTransport / AsyncHttpTransport
         ├── Chọn chiến lược auth       ← API Key / JWT Bearer / x-access-token
         ├── Retry với backoff          ← 429 và 5xx
         └── Connection pool (httpx)   ← keepalive, max_connections=10
               │
               ▼
         Imbrace Gateway               ← app-gatewayv2.imbrace.co
```

### Kiến Trúc Nội Bộ

| Tầng             | File                           | Trách nhiệm                                       |
| ---------------- | ------------------------------ | ------------------------------------------------- |
| Entry Points     | `client.py`, `async_client.py` | Khởi tạo resources, login helpers                 |
| HTTP Transport   | `http.py`                      | Auth headers, retry, timeout, error mapping       |
| Token Manager    | `auth/token_manager.py`        | Đọc/ghi token thread-safe                         |
| Service Registry | `service_registry.py`          | Resolve per-service URL từ env preset             |
| Resources        | `resources/*.py`               | API method wrapper cho từng domain (31 resources) |
| Types            | `types/*.py`                   | Pydantic models và dataclasses                    |
| Exceptions       | `exceptions.py`                | `AuthError`, `ApiError`, `NetworkError`           |

### Sync vs Async — Khi Nào Dùng Cái Nào

| Client               | Framework                      | Import                                   |
| -------------------- | ------------------------------ | ---------------------------------------- |
| `ImbraceClient`      | Django, Flask, scripts, Celery | `from imbrace import ImbraceClient`      |
| `AsyncImbraceClient` | FastAPI, asyncio, aiohttp      | `from imbrace import AsyncImbraceClient` |

:::tip
Cả hai client có interface resource giống hệt nhau — sự khác biệt duy nhất là `await` trên các method calls. Bạn có thể hoán đổi mà không cần thay business logic.
:::
