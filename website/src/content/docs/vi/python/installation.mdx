### Cài Đặt

```bash
pip install imbrace
# hoặc
uv add imbrace
```

Chế độ development (từ source, bao gồm dev tools):

```bash
cd py
pip install -e ".[dev]"
# Cài thêm: pytest, ruff, mypy, pytest-httpx
```

Để cấu hình thông tin xác thực, môi trường và override URL dịch vụ, xem [Hướng Dẫn Cài Đặt](/vi/getting-started/setup/).

---

### Kiểm Tra Cài Đặt

```python
from imbrace import ImbraceClient
client = ImbraceClient()
print("SDK sẵn sàng")
```

---

### Biến Môi Trường

| Biến               | Bắt Buộc | Mặc Định | Mô Tả                                   |
| ------------------ | -------- | -------- | --------------------------------------- |
| `IMBRACE_API_KEY`  | Có\*     | —        | Server-side API Key                     |
| `IMBRACE_ENV`      | Không    | `stable` | Preset: `develop`, `sandbox`, `stable`  |
| `IMBRACE_BASE_URL` | Không    | auto     | Override URL gateway                    |
| `IMBRACE_ORG_ID`   | Không    | —        | Organization ID (cho integration tests) |

\*Hoặc truyền `access_token` trực tiếp cho client-side usage.
