## CRM

### Quản Lý Contacts

```python
# Danh sách contacts
contacts = client.contacts.list(limit=50).get("data", [])

# Lấy một contact
contact = client.contacts.get("contact_id")

# Cập nhật thông tin
client.contacts.update("contact_id", {
    "name":  "Alice B.",
    "email": "alice@example.com",
    "phone": "+84901234567",
})

# Lịch sử hoạt động, bình luận, file
activity = client.contacts.get_activities("conversation_id")
comments = client.contacts.get_comments("contact_id")
files    = client.contacts.get_files("contact_id")
```

### CRM Boards (Pipeline)

Boards là nơi lưu trữ dữ liệu CRM — leads, deals, tasks, hoặc bất kỳ dữ liệu có cấu trúc nào.

```python
# Danh sách boards
boards = client.boards.list().get("data", [])

# Lấy items của một board
leads = client.boards.list_items("board_id", limit=100).get("data", [])

# Tạo item mới
lead = client.boards.create_item("board_id", {
    "fields": {"name": "Acme Corp", "status": "new", "value": 50000}
})

# Cập nhật item
client.boards.update_item("board_id", lead["id"], {
    "fields": {"status": "qualified"}
})

# Tìm kiếm items
results = client.boards.search("board_id", {"query": "enterprise"})

# Xuất CSV
csv = client.boards.export_csv("board_id")
```

### Conversations

Quản lý cuộc hội thoại với khách hàng.

```python
# Tìm kiếm conversations
convs = client.conversations.search(
    business_unit_id="bu_xxx", q="support", limit=20
).get("data", [])

# Conversations chưa xử lý
open_convs = client.conversations.get_outstanding(
    business_unit_id="bu_xxx", limit=50
).get("data", [])

# Cập nhật trạng thái
client.conversations.update_status({
    "conversation_id": "conv_xxx",
    "status": "resolved",
})
```

## AI

### LLM Completions

```python
# Completion đơn
response = client.ai.complete(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "Bạn là trợ lý CRM."},
        {"role": "user",   "content": "Tóm tắt ghi chú khách hàng này: ..."},
    ],
)
print(response["choices"][0]["message"]["content"])

# Streaming
for chunk in client.ai.stream(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Soạn email follow-up cho lead này."}],
):
    print(chunk["choices"][0]["delta"].get("content", ""), end="", flush=True)

# Embeddings
result = client.ai.embed(
    model="text-embedding-ada-002",
    input=["khách hàng khiếu nại về billing", "vấn đề billing leo thang"],
)
```

### Chat AI

Phiên chat có lịch sử, được hỗ trợ bởi dịch vụ aiv2.

models   = client.chat_ai.list_models()

### Hiểu Tài Liệu

Trích xuất dữ liệu có cấu trúc từ PDF, hóa đơn, hình ảnh và các tài liệu khác.

```python
providers = client.chat_ai.list_document_models()

result = client.chat_ai.process_document(
    model_name="gpt-4o",
    url="https://example.com/invoice.pdf",
    organization_id="org_xxx",  # bắt buộc cho endpoint này
)
print(result["data"])
# {"invoice_number": "INV-001", "total": 1200, "vendor": "Acme Corp", ...}
```

:::note
`organization_id` chỉ bắt buộc cho `process_document` — các endpoint khác không cần.
:::

### AI Agent

Chat streaming với knowledge base và quản lý phiên chat end-user.

```python
import uuid

# Stream chat với AI assistant (SSE)
resp = client.ai_agent.stream_chat({
    "id": str(uuid.uuid4()),
    "assistant_id": "assistant_id",
    "organization_id": "org_xxx",
    "messages": [{"role": "user", "content": "Deals nào đã chốt quý này?"}],
})
for line in resp.iter_lines():
    print(line)

# Gợi ý câu hỏi
suggestions = client.ai_agent.get_agent_prompt_suggestion("assistant_id")

# Knowledge base
files      = client.ai_agent.list_embedding_files()
classified = client.ai_agent.classify_file(
    url="https://example.com/product-catalog.pdf",
    mime_type="application/pdf",
)

# Phiên chat end-user
chat = client.ai_agent.create_client_chat({
    "id": str(uuid.uuid4()),
    "message": {"id": str(uuid.uuid4()), "role": "user",
                "parts": [{"type": "text", "text": "Xin chào"}]},
    "assistantId": "assistant_id",
})
messages = client.ai_agent.list_client_messages(chat["id"])
client.ai_agent.delete_client_chat(chat["id"])
```
