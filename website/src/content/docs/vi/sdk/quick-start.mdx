### 1. Khởi tạo client

<AuthOnly mode="access-token">
<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    import { ImbraceClient } from "@imbrace/sdk"

    const client = new ImbraceClient({
      accessToken: "your-access-token",
      baseUrl: "https://app-gatewayv2.imbrace.co",
    })
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import os
    from dotenv import load_dotenv
    from imbrace import ImbraceClient

    load_dotenv()

    client = ImbraceClient(
        access_token=os.environ["IMBRACE_ACCESS_TOKEN"],
        base_url="https://app-gatewayv2.imbrace.co",
    )
    ```
  </TabItem>
</Tabs>
</AuthOnly>

<AuthOnly mode="api-key">
<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    import { ImbraceClient } from "@imbrace/sdk"

    const client = new ImbraceClient({
      apiKey: "your-api-key",
      baseUrl: "https://app-gatewayv2.imbrace.co",
    })
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import os
    from dotenv import load_dotenv

    ```
  </TabItem>
</Tabs>
</AuthOnly>

> Chuyển đổi dropdown **Auth** trong tiêu đề để chuyển đổi giữa các ví dụ API Key và Access Token trên toàn trang. Xem [Xác thực](/sdk/authentication/) để biết nên chọn loại nào.

### 2. Lấy danh sách boards của bạn

Boards là các pipeline CRM — khách hàng tiềm năng, giao dịch, nhiệm vụ, hoặc bất kỳ dữ liệu có cấu trúc nào. Lời gọi này chỉ yêu cầu thông tin xác thực của bạn.

<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    const { data: boards } = await client.boards.list()

    for (const board of boards) {
      console.log(board.id, board.name)
    }
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    boards = client.boards.list()

    for board in boards.get("data", []):
        print(board["id"], board.get("name"))
    ```
  </TabItem>
</Tabs>

Nếu bạn thấy danh sách boards của mình, SDK đã được kết nối và xác thực.
