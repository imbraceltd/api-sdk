Imbrace SDK là client chính thức cho Imbrace Gateway, có sẵn cho **TypeScript** và **Python**. Cả hai SDK đều bao bọc cùng một Gateway API với cùng các namespace tài nguyên, cùng mô hình xác thực và cùng ngữ nghĩa retry/lỗi — hãy chọn ngôn ngữ phù hợp với hệ thống của bạn.

> Chỉ cần chuyển đổi tab ngôn ngữ một lần trên trang này, phần còn lại của tài liệu sẽ ghi nhớ lựa chọn của bạn.

### Tính năng chính

| Tính năng | Chi tiết |
|---|---|
| **An toàn kiểu** | Kiểu TypeScript và gợi ý kiểu Python trên mọi tài nguyên |
| **Hai loại thông tin xác thực** | `apiKey` hoặc `accessToken` — xem [Authentication](/sdk/authentication/) |
| **Tự động thử lại** | Thử lại mã 429 và 5xx với exponential backoff, không cần cấu hình |
| **AI phát trực tiếp** | SSE / async iterator cho `streamChat` và kết quả AI |
| **Bất đồng bộ & đồng bộ (Py)** | `ImbraceClient` (đồng bộ) và `AsyncImbraceClient` (bất đồng bộ) |
| **Hủy bỏ (TS)** | Lan truyền `AbortSignal` để hủy bỏ đang thực thi |

### Xin chào, thế giới

<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    import { ImbraceClient } from "@imbrace/sdk"

    const client = new ImbraceClient({ accessToken: "acc_your_token" })
    const me = await client.platform.getMe()
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    from imbrace import ImbraceClient

    with ImbraceClient(access_token="acc_your_token") as client:
        me = client.platform.get_me()
    ```
  </TabItem>

### Các tài nguyên có sẵn

Mọi namespace đều có trên cả hai SDK. Các phương thức tuân theo quy ước ngôn ngữ — `client.aiAgent.streamChat()` trong TS, `client.ai_agent.stream_chat()` trong Python.

| Namespace | Mục đích |
|---|---|
| `client.aiAgent` / `client.ai_agent` | Chat AI phát trực tiếp, embeddings, parquet, sub-API chat-client |
| `client.chatAi` / `client.chat_ai` | CRUD trợ lý (tạo/cập nhật/xóa/liệt kê trợ lý) |
| `client.workflows` | Tự động hóa quy trình — luồng, trình kích hoạt, lần chạy |
| `client.boards` | Bảng CRM — CRUD, mục, trường, tìm kiếm, phân đoạn, CSV; thư mục & tệp Knowledge Hub |
| `client.platform` | Người dùng, tổ chức, quyền hạn |
| `client.contacts`, `client.conversations`, `client.messages`, `client.channel` | Lớp liên hệ / kênh |
| `client.ai` | Kết quả và embeddings tương thích OpenAI |

Để xem danh sách đầy đủ và tài liệu tham khảo phương thức, hãy xem [Resources](/sdk/resources/).

### Khi nào chọn loại thông tin xác thực nào

| | API Key | Access Token |
|---|---|---|
| **Imbrace là backend của ai?** | Một tính năng bên trong *backend của bạn* | *Imbrace CHÍNH LÀ* backend của bạn |
| **Người dùng của ai?** | Của bạn | Của Imbrace |
| **Phù hợp nhất cho** | Máy chủ đến máy chủ, tập lệnh nội bộ, tích hợp CRM | Ứng dụng hướng đến người dùng, nơi mỗi người dùng cuối đăng nhập |

Cây quyết định đầy đủ: [Authentication →](/sdk/authentication/).
