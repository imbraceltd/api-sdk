### Cài đặt

### Khởi tạo client

<AuthOnly mode="access-token">
<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    import { ImbraceClient } from "@imbrace/sdk"

    // Build on Imbrace — Imbrace IS your backend, end-users log in via OTP
    const client = new ImbraceClient({
      accessToken: "acc_your_token",
      baseUrl: "https://app-gatewayv2.imbrace.co",
    })
    ```

    Client có trạng thái — hãy tạo một lần và tái sử dụng trong toàn bộ ứng dụng.
  </TabItem>
  <TabItem label="Python">
    ```python
    from imbrace import ImbraceClient

    # Build on Imbrace — Imbrace IS your backend, end-users log in via OTP
    with ImbraceClient(access_token="acc_your_token") as client:
        ...
    ```

    Python cũng xuất `AsyncImbraceClient` (`async with ...`) cho các hệ thống async như FastAPI.

    Trình quản lý ngữ cảnh (context manager) tự động đóng kết nối HTTP bên dưới.
  </TabItem>
</Tabs>
</AuthOnly>

### Biến môi trường

SDK **không** tự động đọc biến môi trường. Hãy truyền trực tiếp thông tin xác thực vào hàm tạo (constructor) và sử dụng bộ nạp (`dotenv` / xử lý biến môi trường của framework) nếu bạn lưu chúng trong `.env`.

| Biến | Mục đích |
|---|---|
| `IMBRACE_API_KEY` | Khóa API của bạn (thông tin xác thực cấp tổ chức) |
| `IMBRACE_ACCESS_TOKEN` | Token truy cập của người dùng (thông tin xác thực theo phiên) |
| `IMBRACE_BASE_URL` | Ghi đè URL gateway (mặc định: `https://app-gatewayv2.imbrace.co`) |

Org ID được mã hóa bên trong cả khóa API và token truy cập — bạn không bao giờ cần truyền `organizationId`/`organization_id` cho SDK. Xem [Xác thực](/sdk/authentication/) để biết chi tiết.
