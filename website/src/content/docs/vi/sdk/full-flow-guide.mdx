2. **Tạo AI agent**

   `workflow_name` phải là duy nhất trong tổ chức của bạn.

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const agent = await client.chatAi.createAiAgent({
         name: "Support Bot",
         workflow_name: "support_bot_v1",
         description: "Handles tier-1 customer support queries",
         instructions: "You are a helpful support agent. Be concise and friendly.",
         provider_id: "system",   // use the org's default LLM provider
         model_id: "gpt-4o",      // any model name the system provider exposes
       })

       const assistantId = agent.id  // UUID — use this for all subsequent calls
       console.log("AI agent created:", assistantId)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       agent = client.chat_ai.create_ai_agent({
           "name": "Support Bot",
           "workflow_name": "support_bot_v1",
           "description": "Handles tier-1 customer support queries",
           "instructions": "You are a helpful support agent. Be concise and friendly.",
           "provider_id": "system",   # use the org's default LLM provider
           "model_id": "gpt-4o",      # any model name the system provider exposes
       })

       assistant_id = agent["id"]
       print("AI agent created:", assistant_id)
       ```
     </TabItem>
   </Tabs>

   `provider_id` và `model_id` là bắt buộc. Truyền `provider_id: "system"` để ủy quyền cho nhà cung cấp LLM mặc định của tổ chức hoặc truyền UUID của nhà cung cấp tùy chỉnh. Với `provider_id: "system"`, `model_id` chấp nhận tên mô hình như `"gpt-4o"` hoặc giá trị `"Default"` để dùng mô hình hệ thống mặc định.

3. **Stream phản hồi chat bằng AI agent**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const response = await client.aiAgent.streamChat({
         assistant_id: assistantId,
         messages: [{ role: "user", content: "How do I reset my password?" }],
         // id is the session UUID — reuse it to maintain conversation history
         // If omitted, a new UUID is auto-generated each call
       })

       const reader = response.body!.getReader()
       const decoder = new TextDecoder()

       while (true) {
         const { done, value } = await reader.read()
         if (done) break
         const text = decoder.decode(value)
         for (const line of text.split("\n")) {
           if (line.startsWith("data: ")) {
             const data = line.slice(6).trim()
             if (data && data !== "[DONE]") {
               try {
                 const chunk = JSON.parse(data)
                 process.stdout.write(chunk.delta ?? chunk.content ?? "")
               } catch {}
             }
           }
         }
       }
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       response = client.ai_agent.stream_chat({
           "assistant_id": assistant_id,
           "messages": [{"role": "user", "content": "How do I reset my password?"}],
           # id is the session UUID — reuse it to maintain conversation history.
           # If omitted, a new UUID is auto-generated each call.
       })

       import json
       for line in response.iter_lines():
           if isinstance(line, bytes):
               line = line.decode()
           if not line.startswith("data: "):
               continue
           data = line[6:].strip()
           if data and data != "[DONE]":
               try:
                   chunk = json.loads(data)
                   print(chunk.get("delta") or chunk.get("content") or "", end="")
               except Exception:
                   pass
       ```
     </TabItem>
   </Tabs>

4. **Duy trì lịch sử hội thoại (session ID)**

   Truyền cùng `id` (phải là UUID) qua các lần gọi để giữ ngữ cảnh:

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       import { randomUUID } from "crypto"

       const sessionId = randomUUID()

       // First message
       await client.aiAgent.streamChat({
         assistant_id: assistantId,
         id: sessionId,
         messages: [{ role: "user", content: "What's your refund policy?" }],
       })

       // Follow-up — same session, assistant remembers context
       await client.aiAgent.streamChat({
         assistant_id: assistantId,
         id: sessionId,
         messages: [{ role: "user", content: "How long does it take?" }],
       })
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       import uuid

       session_id = str(uuid.uuid4())

       # First message
       client.ai_agent.stream_chat({
           "assistant_id": assistant_id,
           "id": session_id,
           "messages": [{"role": "user", "content": "What's your refund policy?"}],
       })

       # Follow-up — same session, assistant remembers context
       client.ai_agent.stream_chat({
           "assistant_id": assistant_id,
           "id": session_id,
           "messages": [{"role": "user", "content": "How long does it take?"}],
       })
       ```
     </TabItem>
   </Tabs>

## 2. Tạo Workflow và gắn nó vào AI Agent

<Steps>

1. **Liệt kê các flow hiện có để tìm project ID của bạn**

   <Tabs syncKey="lang">

       ```typescript
       const projectId = flows[0]?.projectId
       console.log("Project ID:", projectId)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       res = client.workflows.list_flows(limit=5)
       flows = res.get("data", [])
       project_id = flows[0]["projectId"] if flows else None
       print("Project ID:", project_id)
       ```
     </TabItem>
   </Tabs>

2. **Tạo flow mới**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const flow = await client.workflows.createFlow({
         displayName: "CRM Update on New Lead",

       })
       console.log("Flow created:", flow.id)
       ```

     </TabItem>
     <TabItem label="Python">

       ```python
       print("Flow created:", flow["id"])
       ```
     </TabItem>
   </Tabs>

3. **Thêm trigger Webhook và xuất bản flow**

   Flow mới tạo ở trạng thái **DRAFT** không có trigger — URL webhook chưa tồn tại, vì vậy `triggerFlow` sẽ trả về 404. Thêm piece Webhook làm trigger, sau đó xuất bản:

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       // Set the Webhook piece as the flow's trigger
       await client.workflows.applyFlowOperation(flow.id, {
         type: "UPDATE_TRIGGER",
         request: {
           name: "trigger",
           type: "PIECE_TRIGGER",
           valid: true,
           displayName: "Webhook",
           settings: {
             pieceName: "@activepieces/piece-webhook",
             pieceVersion: "0.1.24",
             triggerName: "catch_webhook",
             input: { authType: "none" },
             propertySettings: {},
           },
         },
       })

       // Publish — flow status flips DISABLED → ENABLED and webhook URL becomes live
       await client.workflows.applyFlowOperation(flow.id, {
         type: "LOCK_AND_PUBLISH",
         request: {},
       })
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       # Set the Webhook piece as the flow's trigger
       client.workflows.apply_flow_operation(flow["id"], {
           "type": "UPDATE_TRIGGER",
           "request": {
               "name": "trigger",
               "type": "PIECE_TRIGGER",
               "valid": True,
               "displayName": "Webhook",
               "settings": {
                   "pieceName": "@activepieces/piece-webhook",
                   "pieceVersion": "0.1.24",
                   "triggerName": "catch_webhook",
                   "input": {"authType": "none"},
                   "propertySettings": {},
               },
           },
       })

       # Publish — DISABLED → ENABLED, webhook URL becomes live
       client.workflows.apply_flow_operation(flow["id"], {
           "type": "LOCK_AND_PUBLISH",
           "request": {},
       })
       ```
     </TabItem>
   </Tabs>

4. **Kích hoạt flow thủ công với payload**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       // Fire and forget (async)
       await client.workflows.triggerFlow(flow.id, {
         contact_name: "Jane Smith",
         email: "jane@example.com",
       })

       // Sync trigger — for this to actually return data instead of timing out,
       // the flow needs a "Return Response" action added via applyFlowOperation
       // ADD_ACTION (otherwise the gateway waits 30s for a response and gives up).
       const result = await client.workflows.triggerFlowSync(flow.id, {
         contact_name: "Jane Smith",
         email: "jane@example.com",
       })
       console.log("Flow result:", result)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       # Fire and forget (async)
       client.workflows.trigger_flow(flow["id"], {
           "contact_name": "Jane Smith",
           "email": "jane@example.com",
       })

       # Sync trigger — for this to actually return data instead of timing out,
       # the flow needs a "Return Response" action added via apply_flow_operation
       # ADD_ACTION (otherwise the gateway waits 30s and gives up).
       result = client.workflows.trigger_flow_sync(flow["id"], {
           "contact_name": "Jane Smith",
           "email": "jane@example.com",
       })
       print("Flow result:", result)
       ```
     </TabItem>
   </Tabs>

       ```python
           "name": "Support Bot",
           "workflow_name": "support_bot_v1",

               {"flow_id": flow["id"], "description": "Update CRM on new lead"}
       ```

## 3. Quản lý Knowledge Hub và gắn vào AI Agent

Các tệp và thư mục Knowledge Hub nằm trong dịch vụ **data-board** (`client.boards`). `_id` của thư mục là giá trị bạn truyền cho AI agent làm nguồn kiến thức.

<Steps>

1. **Tạo thư mục**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const folder = await client.boards.createFolder({
         name: "Product Documentation",
         organization_id: "org_your_org_id",
         parent_id: "root",
       })
       console.log("Folder ID:", folder._id)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       folder = client.boards.create_folder({
           "name": "Product Documentation",
           "organization_id": "org_your_org_id",
           "parent_id": "root",
       })
       print("Folder ID:", folder["_id"])
       ```
     </TabItem>
   </Tabs>

2. **Tải tệp lên thư mục**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       import { readFileSync } from "fs"

       const fileBuffer = readFileSync("./docs/faq.pdf")
       const formData = new FormData()
       formData.append("file", new Blob([fileBuffer], { type: "application/pdf" }), "faq.pdf")
       formData.append("folder_id", folder._id)
       formData.append("organization_id", "org_your_org_id")

       const uploaded = await client.boards.uploadFile(formData)
       console.log("File uploaded:", uploaded.file_id)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       from pathlib import Path

       path = Path("./docs/faq.pdf")
       files = {
           "file": (path.name, path.read_bytes(), "application/pdf"),
           "folder_id": (None, folder["_id"]),
           "organization_id": (None, "org_your_org_id"),
       }
       uploaded = client.boards.upload_file(files)
       print("File uploaded:", uploaded.get("file_id") or uploaded.get("_id"))
       ```
     </TabItem>
   </Tabs>

3. **Gắn thư mục vào AI agent**

   Truyền `_id` của thư mục trong `folder_ids` — AI agent truy xuất từ mọi tệp trong thư mục đó. Sử dụng `board_ids` để gắn data-board CRM (các mục của nó trở thành nguồn kiến thức — xem [Tài nguyên → Boards & items](/sdk/resources/#boards--items--clientboards)). Trường `knowledge_hubs` cũ đã không còn được dùng nữa.

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       await client.chatAi.updateAiAgent(assistantId, {
         name: "Support Bot",
         workflow_name: "support_bot_v1",
         folder_ids: [folder._id],
         // board_ids: [boardId],  // optional: attach a CRM data-board too
       })
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       client.chat_ai.update_ai_agent(assistant_id, {
           "name": "Support Bot",
           "workflow_name": "support_bot_v1",
           "folder_ids": [folder["_id"]],
           # "board_ids": [board_id],  # optional: attach a CRM data-board too
       })
       ```
     </TabItem>
   </Tabs>

4. **Kiểm tra và quản lý thư mục và tệp**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript

       const folders = await client.boards.searchFolders({ q: "Product" })

       // Delete folders
       ```

     </TabItem>
     <TabItem label="Python">

       ```python
       # Get folder with contents
       contents = client.boards.get_folder_contents(folder["_id"])
       print("Files:", len(contents.get("files") or []))

       # Rename a folder
       client.boards.update_folder(folder["_id"], {"name": "Product Docs v2"})

       # Search files in a folder

       # Delete folders
       client.boards.delete_folders([folder["_id"]])
       ```
     </TabItem>
   </Tabs>

</Steps>

## 4. Quản lý Data Board và Items (Pipeline CRM)

<Steps>

1. **Tạo board**

   Board là một pipeline CRM — khách hàng tiềm năng, thỏa thuận, nhiệm vụ hoặc bất kỳ dữ liệu có cấu trúc nào.

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const board = await client.boards.create({
         name: "Sales Pipeline",
         description: "Track all active deals",
       })
       console.log("Board ID:", board._id)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       board = client.boards.create(
           name="Sales Pipeline",
           description="Track all active deals",
       )
       print("Board ID:", board["_id"])
       ```
     </TabItem>
   </Tabs>

2. **Thêm trường tùy chỉnh**

   Các loại trường là `ShortText`, `LongText`, `Number`, `Dropdown`, `Date`, `Checkbox`, v.v. `createField` trả về board đã cập nhật — tìm trường mới của bạn bên trong `board.fields`.

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const updated = await client.boards.createField(board._id, {
         name: "Company",
         type: "ShortText",
       })
       // Find the identifier field (auto-created with every board)
       const identifierField = updated.fields.find(f => f.is_identifier)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       updated = client.boards.create_field(board["_id"], {
           "name": "Company",
           "type": "ShortText",
       })
       identifier_field = next(f for f in updated["fields"] if f.get("is_identifier"))
       ```
     </TabItem>
   </Tabs>

3. **Tạo mục board (bản ghi)**

   Các mục sử dụng định dạng `{ fields: [{ board_field_id, value }] }`:

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       const item = await client.boards.createItem(board._id, {
         fields: [
           { board_field_id: identifierField._id, value: "Acme Corp" },
         ],
       })
       console.log("Item ID:", item._id)
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       item = client.boards.create_item(board["_id"], {
           "fields": [
               {"board_field_id": identifier_field["_id"], "value": "Acme Corp"},
           ],
       })
       print("Item ID:", item["_id"])
       ```
     </TabItem>
   </Tabs>

4. **Liệt kê và tìm kiếm mục**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">
       ```typescript
       // Paginate through items
       const { data: items } = await client.boards.listItems(board._id, { limit: 20, skip: 0 })

       // Full-text search
       const { data: results } = await client.boards.search(board._id, {
         q: "Acme",
         limit: 10,
       })
       ```
     </TabItem>
     <TabItem label="Python">
       ```python
       # Paginate through items
       items = client.boards.list_items(board["_id"], limit=20, skip=0)

       # Full-text search
       results = client.boards.search(board["_id"], q="Acme", limit=10)
       ```
     </TabItem>
   </Tabs>

5. **Cập nhật và xóa mục**

   <Tabs syncKey="lang">
     <TabItem label="TypeScript">

   </Tabs>
