## Cài đặt CLI

### Gói npm công khai

```bash
npm install -g @imbrace/cli
```

### Từ mã nguồn

Clone kho lưu trữ và chạy script cài đặt:

```bash
git clone https://github.com/imbraceltd/imbrace-cli.git
cd imbrace-cli
./install.sh
```

`install.sh` chạy `npm install`, `npm run build`, `npm link`, sau đó tạo symlink `imbrace` vào `/opt/homebrew/bin` (Apple Silicon) hoặc `/usr/local/bin` (Intel).

### Biên dịch thủ công

```bash
cd cli
npm install
npm run build
npm link
```

## Xác thực

Tất cả các lệnh (ngoại trừ `login`) sẽ tự động yêu cầu đăng nhập nếu chưa được xác thực.

### Đăng nhập với API Key

Khuyến nghị cho CI/CD và các tác nhân lập trình:

```bash
imbrace login --api-key api_xxx...
```

### Đăng nhập với Email + Mật khẩu

```bash
imbrace login --email user@example.com --password mypass
```

### Kiểm tra đăng nhập hiện tại

```bash
imbrace whoami [--json]
```

### Đăng xuất

```bash
imbrace logout
```

<Aside type="tip">
  Nếu một lệnh trả về 401, hãy chạy `imbrace login --api-key api_xxx...` lại.
</Aside>

## Lưu trữ thông tin xác thực

| Hệ điều hành | Đường dẫn |
|---|---|
| macOS | `~/Library/Preferences/imbrace-nodejs/config.json` |
| Linux | `~/.config/imbrace-nodejs/config.json` |
| Windows | `%APPDATA%\imbrace-nodejs\Config\config.json` |
