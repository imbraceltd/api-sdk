**Vibe Coding** là cách viết code bằng cách cộng tác với AI assistant — mô tả yêu cầu bằng ngôn ngữ tự nhiên và để AI tạo, giải thích, hoặc refactor code cho bạn. Imbrace SDK đi kèm file [`llms.txt`](https://developer.imbrace.co/llms.txt) để bất kỳ AI tool nào cũng có thể hiểu SDK ngay lập tức mà không bịa ra tên method hay shape của argument.

## Thiết lập

Trước khi bắt đầu vibe code, hãy đảm bảo SDK đã được cài đặt và credentials đã được cấu hình.

**1. Cài đặt SDK**

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```bash
npm install @imbrace/sdk
```
</TabItem>
<TabItem label="Python">
```bash
pip install imbrace
```
</TabItem>
</Tabs>

**2. Cấu hình credentials**

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```bash
export IMBRACE_API_KEY=your_api_key
export IMBRACE_ORG_ID=your_org_id
```
</TabItem>
<TabItem label="Python">
```bash
export IMBRACE_API_KEY=your_api_key
export IMBRACE_ORG_ID=your_org_id
```
</TabItem>
</Tabs>

Xem [Authentication](/sdk/authentication/) để biết cách lấy API key.

**3. Lấy file `llms.txt`**

Tải hoặc copy file tại [`https://developer.imbrace.co/llms.txt`](https://developer.imbrace.co/llms.txt) và đưa vào AI tool của bạn (xem [Cách sử dụng](#cách-sử-dụng) bên dưới).

## `llms.txt` là gì?

`llms.txt` là file plain-text (tương tự `robots.txt`) cung cấp cho AI model bản tóm tắt chính xác, gọn nhẹ về thư viện — clients, resources, authentication, và các pattern phổ biến. Khi bạn paste nó vào context window của AI, model đã hiểu SDK và có thể viết code đúng ngay từ lần đầu.

**URL file:** [`https://developer.imbrace.co/llms.txt`](https://developer.imbrace.co/llms.txt)

## Cách sử dụng

### Claude (claude.ai hoặc Claude Code)

1. Mở conversation mới.
2. Paste nội dung `llms.txt` vào đầu tin nhắn, rồi mô tả task của bạn:

```
<context>
[paste llms.txt vào đây]
</context>

Viết TypeScript snippet stream chat response từ assistant "asst_abc"
và in từng text delta ra console.
```

### Cursor / VS Code Copilot

Thêm URL vào AI context qua tính năng **@ docs** hoặc "add context" trong IDE của bạn. Cursor hỗ trợ `@URL` trực tiếp:

```
@https://developer.imbrace.co/llms.txt

Làm thế nào để upload file và trigger embedding processing?
```

### Các LLM khác

Copy nội dung raw của file và paste vào đầu prompt trước khi đặt câu hỏi. Hầu hết LLM với context window 32k+ đều có thể đọc toàn bộ file mà không bị mất thông tin.

## Ví dụ prompt

Sau khi AI có context từ `llms.txt`, thử các prompt như:

- *"Hướng dẫn tạo AI agent và stream chat response bằng Python."*
- *"Tạo TypeScript code để list tất cả embedding file và xóa những file có status `error`."*
- *"Sự khác nhau giữa `streamChat` và `streamSubAgentChat` là gì?"*
- *"Viết Express.js auth proxy cho Chat Client theo pattern trong Integrations guide."*

<Aside type="tip">
Với những session dài, hãy pin `llms.txt` làm **system prompt** hoặc **project instruction** để nó tự động áp dụng cho mọi tin nhắn.
</Aside>

### CLI cho Coding Agents

Imbrace CLI (`@imbrace/cli`) cũng được thiết kế cho coding agent. Sau khi cài đặt CLI, AI assistant có thể chạy lệnh trực tiếp qua terminal mà không cần viết mã SDK:

```bash
# Danh sách boards
imbrace data-board list --json

# Tạo items
imbrace data-board create-item <boardId> --fields '<json>' --json

# Danh sách AI agents
imbrace ai-agent list --json

# Quản lý workflows
imbrace workflow list --json
```

Tất cả lệnh đều hỗ trợ `--json` cho đầu ra máy đọc được và `-h` / `--help` để xem hướng dẫn. Xem [CLI Commands](/cli/commands/) để tham khảo đầy đủ.

## Cập nhật định kỳ

File được tạo lại sau mỗi lần release. Hãy fetch lại URL khi bạn nâng cấp SDK để cập nhật các method mới hoặc signature đã thay đổi.
