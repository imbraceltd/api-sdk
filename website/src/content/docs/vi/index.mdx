## Tại sao nên dùng Imbrace SDK?

<CardGrid>
  <Card title="An toàn kiểu dữ liệu" icon="approve-check-circle">
    Đầy đủ định nghĩa TypeScript và type hint Python trên tất cả 27+ resource
    namespaces. Phát hiện lỗi ở thời điểm biên dịch, không phải lúc chạy.
  </Card>
  <Card title="Tự động thử lại & Phục hồi" icon="random">
    Exponential backoff tích hợp sẵn cho các phản hồi 429 và 5xx. Không cần cấu
    hình — ứng dụng của bạn luôn hoạt động dưới tải.
  </Card>
  <Card title="Xác thực 3 tầng" icon="setting">
    API Key, JWT Bearer, và access token cũ — tất cả đều được xử lý minh bạch.
    Chuyển đổi chiến lược xác thực mà không cần thay đổi logic nghiệp vụ.
  </Card>
  <Card title="Async-first Python" icon="rocket">
    `AsyncImbraceClient` song song cùng với client đồng bộ. Tích hợp vào FastAPI,
    asyncio, hoặc Django chỉ với một import.
  </Card>
</CardGrid>

</div>

## Cài đặt nhanh

<div class="install-grid">
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--ts">TypeScript</span>
    </div>

    ```bash
    npm install @imbrace/sdk
    ```

    ```ts
    import { ImbraceClient } from "@imbrace/sdk";
    const client = new ImbraceClient();
    const me = await client.platform.getMe();
    ```

  </div>
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--py">Python</span>
    </div>

    ```bash
    pip install imbrace
    ```

    ```python
    from imbrace import ImbraceClient
    client = ImbraceClient()
    me = client.platform.get_me()
    ```

  </div>
</div>
