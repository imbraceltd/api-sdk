Imbrace is a customer engagement platform — it handles chat channels, CRM pipelines, AI assistants, automated workflows, and document processing in one place. The SDK lets you embed those features directly into your own applications without building the infrastructure yourself.

### What You Can Build

- **AI chat assistants** — stream live responses from an AI assistant trained on your knowledge base ([Full Flow Guide §1](/typescript/full-flow-guide/#1-create-an-ai-assistant-and-start-chatting))
- **Automated workflows** — trigger and manage multi-step automations via Activepieces ([Full Flow Guide §2](/typescript/full-flow-guide/#2-create-a-workflow-with-activepieces-and-bind-it-to-an-assistant))
- **Knowledge Hub** — upload files, attach folders to assistants for RAG ([Full Flow Guide §3](/typescript/full-flow-guide/#3-manage-knowledge-hubs-and-attach-to-an-assistant))
- **CRM pipelines** — create boards, manage leads, search and export deal data ([Full Flow Guide §4](/typescript/full-flow-guide/#4-manage-data-boards-and-items-crm-pipelines))

### Available SDKs

| SDK | Package | Runtime |
|---|---|---|
| TypeScript / JavaScript | `@imbrace/sdk` | Node.js 18+, browser |
| Python | `imbrace` | Python 3.9+ |

### Authentication

Two credential types: **API Key** (service-to-service) and **Access Token** (per-user session). See [Authentication](/typescript/authentication/) for which to pick and why.

### Quick Example

```typescript
import { ImbraceClient } from "@imbrace/sdk"

const client = new ImbraceClient({
  accessToken: "your-access-token",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})

const stream = await client.aiAgent.streamChat({
  assistant_id: "asst_xxx",
  organization_id: "org_xxx",
  messages: [{ role: "user", content: "What deals closed this quarter?" }],
})
```
