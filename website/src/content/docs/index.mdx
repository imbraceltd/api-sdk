## Why Imbrace SDK?

<CardGrid>
  <Card title="Type-safe by Design" icon="approve-check-circle">
    Full TypeScript definitions and Python type hints across all 27+ resource
    namespaces. Catch errors at compile time, not runtime.
  </Card>
  <Card title="Auto-retry & Resilience" icon="random">
    Built-in exponential backoff for 429 and 5xx responses. Zero configuration
    needed — your app stays alive under load.
  </Card>
  <Card title="Two Credential Types" icon="setting">
    **API Key** for service-to-service, **Access Token** for per-user
    sessions. The transport handles headers automatically.
    [Learn when to use each →](typescript/authentication/)
  </Card>
  <Card title="Async-first Python" icon="rocket">
    Parallel `AsyncImbraceClient` alongside the sync client. Drop into FastAPI,
    asyncio, or Django with one import.
  </Card>
</CardGrid>

</div>

## Quick Install

<div class="install-grid">
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--ts">TypeScript</span>
    </div>

    ```bash
    npm install @imbrace/sdk
    ```

    ```ts
    import { ImbraceClient } from "@imbrace/sdk";
    const client = new ImbraceClient();
    const me = await client.platform.getMe();
    ```

  </div>
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--py">Python</span>
    </div>

    ```bash
    pip install imbrace
    ```

    ```python
    from imbrace import ImbraceClient
    client = ImbraceClient()
    me = client.platform.get_me()
    ```

  </div>
</div>
