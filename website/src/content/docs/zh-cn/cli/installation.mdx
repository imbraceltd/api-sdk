## 安装 CLI

### 公共 npm 包

```bash
npm install -g @imbrace/cli
```

### 从源码安装

克隆仓库并运行安装脚本：

```bash
git clone https://github.com/imbraceltd/imbrace-cli.git
cd imbrace-cli
./install.sh
```

`install.sh` 依次运行 `npm install`、`npm run build`、`npm link`，然后将 `imbrace` 符号链接到 `/opt/homebrew/bin`（Apple Silicon）或 `/usr/local/bin`（Intel）。

### 手动构建

```bash
cd cli
npm install
npm run build
npm link
```

## 认证

所有命令（`login` 除外）在未认证时会自动提示登录。

### 使用 API Key 登录

推荐用于 CI/CD 和编码代理：

```bash
imbrace login --api-key api_xxx...
```

### 使用邮箱 + 密码登录

```bash
imbrace login --email user@example.com --password mypass
```

### 查看当前登录状态

```bash
imbrace whoami [--json]
```

### 退出登录

```bash
imbrace logout
```

<Aside type="tip">
  如果命令返回 401，请重新运行 `imbrace login --api-key api_xxx...`。
</Aside>

## 凭证存储

| 操作系统 | 路径 |
|---|---|
| macOS | `~/Library/Preferences/imbrace-nodejs/config.json` |
| Linux | `~/.config/imbrace-nodejs/config.json` |
| Windows | `%APPDATA%\imbrace-nodejs\Config\config.json` |
