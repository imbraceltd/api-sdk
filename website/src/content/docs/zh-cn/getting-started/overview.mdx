Imbrace 是一个客户互动平台 — 它在同一个地方处理聊天频道、CRM 管道、AI 代理、自动化工作流和文档处理。SDK 让您可以直接将这些功能嵌入到自己的应用程序中，无需自行构建基础设施。

### 您可以构建什么

- **AI 代理** — 从基于您的知识库训练的 AI 代理实时流式传输响应
- **自动化工作流** — 通过工作流引擎触发和管理多步骤自动化
- **CRM 管道** — 创建看板、管理潜在客户、搜索和导出交易数据
- **文档处理** — 从 PDF 和发票中提取结构化数据，生成嵌入向量

### CLI 工具

Imbrace 还提供了一个命令行界面（`@imbrace/cli`），用于从终端与平台交互。它专为开发者和编码代理设计 — 与 SDK 一起安装，用于快速原型设计、调试和自动化。

详情请参阅 [CLI 概述](/cli/overview/)。

### 可用的 SDK

| SDK | 包 | 运行时 |
|---|---|---|
| TypeScript / JavaScript | `@imbrace/sdk` | Node.js 18+, 浏览器 |
| Python | `imbrace` | Python 3.9+ |

### 身份验证

有两种验证方式：

- **访问令牌** — 用于面向用户的应用程序。用户在通过 OTP 或 OAuth 登录后获取。在初始化客户端时作为 `accessToken` 传入。
- **API 密钥** — 用于服务器到服务器的集成。从 Imbrace 管理仪表板生成。作为 `apiKey` 传入。

### 网关

所有 SDK 请求都通过：

```
https://app-gatewayv2.imbrace.co
```

网关默认设置 — 除非您针对不同环境，否则无需配置。

### 快速示例

```typescript
import { ImbraceClient } from "@imbrace/sdk"

const client = new ImbraceClient({
  accessToken: "your-access-token",
  baseUrl: "https://app-gatewayv2.imbrace.co",
})

const stream = await client.aiAgent.streamChat({
  assistant_id: "asst_xxx",
  messages: [{ role: "user", content: "What deals closed this quarter?" }],
})
```
