Imbrace SDK 将通信组织为三个密切相关的资源：

| 资源 | Client namespace | 说明 |
|------|-----------------|------|
| [Channel](/reference/channel/) | `client.channel` | 消息渠道（Web、WhatsApp、Facebook、Instagram、Email、WeChat、LINE） |
| [Conversation](/reference/conversation/) | `client.conversations` | 渠道内的单个对话线程 |
| [Contact](/reference/contact/) | `client.contacts` | 链接到对话的联系人（潜在客户、客户） |

---

## 典型流程

1. **创建 Channel** — 设置用户将通过其通信的消息渠道。
2. **对话会自动创建** — 当联系人在该渠道上发起消息时，或者您也可以通过编程方式创建。
3. **Contacts** 与对话关联，并存储身份信息（姓名、电子邮件、电话）。

请参阅每个资源的参考页面以获取完整的 schema 和方法详细信息。
