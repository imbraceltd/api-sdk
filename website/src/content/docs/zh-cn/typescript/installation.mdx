---

### 安装

```bash
npm install @imbrace/sdk
# 或
yarn add @imbrace/sdk
# 或
pnpm add @imbrace/sdk
```

凭证、环境和服务 URL 覆盖的配置，请参阅[安装指南](/zh-cn/getting-started/setup/)。

---

### 构建脚本

```bash
npm run build      # 编译 TypeScript → dist/
npm run dev        # 监视模式 — 文件变更时自动重新构建
npm run typecheck  # 仅类型检查（不生成文件）
npm run test       # 运行单元测试
npm run clean      # 清除 dist/
```

---

### 环境变量参考

| 变量                  | 必填 | 默认值   | 说明                                             |
| --------------------- | ---- | -------- | ------------------------------------------------ |
| `IMBRACE_API_KEY`     | 是\* | —        | 服务端 API Key                                   |
| `IMBRACE_ENV`         | 否   | `stable` | 预设：`develop`、`sandbox`、`stable`              |
| `IMBRACE_BASE_URL`    | 否   | 自动     | 覆盖网关 URL（优先于 `IMBRACE_ENV`）              |

\*或直接传入 `accessToken` 用于客户端使用。
