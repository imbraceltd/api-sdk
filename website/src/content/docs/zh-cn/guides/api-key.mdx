import { Tabs, TabItem } from "@astrojs/starlight/components";

API Key（`api_...`）允许你的后端在无需用户会话的情况下调用 Imbrace。关于何时使用 API Key 而非 Access Token，请参阅[身份验证](/zh-cn/sdk/authentication/)。

---

## 方式一 — Imbrace Portal（推荐）

1. 登录 Imbrace Portal。
2. 进入 **Settings → API Keys**。
3. 点击 **Create** 并为 Key 命名。
4. 复制 `api_...` 值 — 仅显示一次。

---

## 方式二 — SDK（编程方式）

适用于需要从代码生成 Key 的场景，例如自动化部署。需要先通过 [OTP](/zh-cn/sdk/authentication/#otp-login-flow) 或[密码](/zh-cn/sdk/authentication/#password-login)登录获取 Access Token。

<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    // 需要使用 access token 初始化 client
    const res = await client.auth.getThirdPartyToken(30) // 30 天后过期
    const apiKey = res.apiKey.apiKey  // "api_..."
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    # 需要使用 access token 初始化 client
    res = client.auth.get_third_party_token(expiration_days=30)
    api_key = res["apiKey"]["apiKey"]  # "api_..."
    ```
  </TabItem>
</Tabs>

返回结构：

```json
{
  "apiKey": {
    "apiKey": "api_...",
    "expired_at": "2025-08-01T00:00:00.000Z",
    "is_active": true
  },
  "expires_in": 2592000
}
```

---

## 使用 Key

将 Key 传入 client 构造函数：

<Tabs syncKey="lang">
  <TabItem label="TypeScript">
    ```typescript
    import { ImbraceClient } from "@imbrace/sdk";

    const client = new ImbraceClient({
      apiKey: process.env.IMBRACE_API_KEY!,
      baseUrl: "https://app-gatewayv2.imbrace.co",
    });
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import os
    from imbrace import ImbraceClient

    client = ImbraceClient(api_key=os.environ["IMBRACE_API_KEY"])
    ```
  </TabItem>
</Tabs>
