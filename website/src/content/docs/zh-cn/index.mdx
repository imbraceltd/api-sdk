## 为什么选择 Imbrace SDK？

<CardGrid>
  <Card title="类型安全设计" icon="approve-check-circle">
    覆盖 27+ 资源命名空间的完整 TypeScript 定义和 Python 类型注解。在编译时捕获错误，而非运行时。
  </Card>
  <Card title="自动重试与弹性" icon="random">
    针对 429 和 5xx 响应内置指数退避重试。无需配置 — 高负载下应用依然稳定运行。
  </Card>
  <Card title="三层身份验证" icon="setting">
    API Key、JWT Bearer 和 legacy access token — 全部透明处理。无需更改业务逻辑即可切换认证策略。
  </Card>
  <Card title="Python 异步优先" icon="rocket">
    `AsyncImbraceClient` 与同步客户端并行存在。一行导入即可接入 FastAPI、asyncio 或 Django。
  </Card>
</CardGrid>

</div>

## 快速安装

<div class="install-grid">
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--ts">TypeScript</span>
    </div>

    ```bash
    npm install @imbrace/sdk
    ```

    ```ts
    import { ImbraceClient } from "@imbrace/sdk";
    const client = new ImbraceClient();
    const me = await client.platform.getMe();
    ```

  </div>
  <div class="install-card">
    <div class="install-card__header">
      <span class="install-badge install-badge--py">Python</span>
    </div>

    ```bash
    pip install imbrace
    ```

    ```python
    from imbrace import ImbraceClient
    client = ImbraceClient()
    me = client.platform.get_me()
    ```

  </div>
