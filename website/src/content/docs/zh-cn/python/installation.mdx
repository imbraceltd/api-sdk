### 安装

```bash
pip install imbrace
# 或
uv add imbrace
```

开发模式（从源码安装，包含开发工具）：

```bash
cd py
pip install -e ".[dev]"
# 额外安装：pytest, ruff, mypy, pytest-httpx
```

如需配置认证信息、环境和服务 URL 覆盖，请参阅[安装指南](/zh-cn/getting-started/setup/)。

---

### 验证安装

```python
from imbrace import ImbraceClient
client = ImbraceClient()
print("SDK 就绪")
```

---

### 环境变量

| 变量               | 必填  | 默认值   | 描述                                    |
| ------------------ | ----- | -------- | --------------------------------------- |
| `IMBRACE_API_KEY`  | 是\*  | —        | 服务端 API Key                          |
| `IMBRACE_ENV`      | 否    | `stable` | 预设：`develop`、`sandbox`、`stable`    |
| `IMBRACE_BASE_URL` | 否    | 自动     | 覆盖 gateway URL                        |
| `IMBRACE_ORG_ID`   | 否    | —        | Organization ID（用于集成测试）         |

\*或直接向客户端传入 `access_token` 用于客户端使用。
