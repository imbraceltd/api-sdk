## AI 代理 — `chatAi` / `chat_ai`

管理 AI 代理（CRUD）、运行 OpenAI 兼容的补全以及处理文档/文件。同一命名空间还涵盖知识库文件夹和知识库。

## OpenAI 兼容 AI 服务 — `client.ai`

OpenAI 风格的补全、流式和嵌入向量。还暴露了 LLM 模型、提供商、护栏、RAG 文件和 AI 代理的管理 API。TypeScript 和 Python 均可用。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
// 单次补全
const response = await client.ai.complete({
  model: "gpt-4o",
  messages: [
    { role: "system", content: "你是一个乐于助人的 CRM 助手。" },
    { role: "user",   content: "总结一下这条客户备注：..." },
  ],
  temperature: 0.7,
});
console.log(response.choices[0].message.content);

// 流式
const stream = client.ai.stream({
  model: "gpt-4o",
  messages: [{ role: "user", content: "为此线索起草一封跟进邮件。" }],
});
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0].delta.content ?? "");
}

// 嵌入向量
const result = await client.ai.embed({
  model: "text-embedding-ada-002",
  input: ["客户投诉了账单问题", "账单问题已升级"],
});
```
</TabItem>
<TabItem label="Python">
```python
from imbrace.types.ai import CompletionInput, CompletionMessage, EmbeddingInput

# 单次补全
response = client.ai.complete(CompletionInput(
    model="gpt-4o",
    messages=[
        CompletionMessage(role="system", content="你是一个乐于助人的 CRM 助手。"),
        CompletionMessage(role="user", content="总结一下这条客户备注：..."),
    ],
    temperature=0.7,
))
print(response.choices[0].message.content)

# 流式
for chunk in client.ai.stream(CompletionInput(
    model="gpt-4o",
    messages=[CompletionMessage(role="user", content="为此线索起草一封跟进邮件。")],
)):
    print(chunk.choices[0].delta.content or "", end="", flush=True)

# 嵌入向量
result = client.ai.embed(EmbeddingInput(
    model="text-embedding-ada-002",
    input=["客户投诉了账单问题", "账单问题已升级"],
))
```
</TabItem>
</Tabs>

## Chat v2 — 流式（SSE）

返回原始响应。将响应体作为服务器发送事件流消费。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const response = await client.aiAgent.streamChat({
  id: "chat_id",
  assistant_id: "asst_abc",
  messages: [{ role: "user", content: "你能做什么？" }],
});

const reader = response.body!.getReader();
const decoder = new TextDecoder();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  process.stdout.write(decoder.decode(value));
}
```
</TabItem>
<TabItem label="Python">
```python
response = client.ai_agent.stream_chat({
    "id": "chat_id",
    "assistant_id": "asst_abc",
    "messages": [{"role": "user", "content": "你能做什么？"}],
})

for line in response.iter_lines():
    if line:
        print(line)
```

**异步：**

```python
import asyncio
from imbrace import AsyncImbraceClient
```

```typescript
const chat = await client.aiAgent.getChat("chat_id", true);

  organization_id: "org_abc",
```
