import { Tabs, TabItem } from "@astrojs/starlight/components";

文档 AI 使用视觉模型（例如 GPT-4o）从 PDF 和图片中提取结构化数据——发票、表单、收据、合同或任何带有可见文字的文档。

SDK 暴露两个命名空间：
- **`client.chatAi` / `client.chat_ai`**——直接的文档处理和模型列表
- **`client.documentAi` / `client.document_ai`**——基于代理的文档处理，包含模式管理、代理 CRUD 和端到端编排

首先初始化客户端（参见[安装](/sdk/installation/)或[快速入门](/sdk/quick-start/)）。

#### PDF 表单填充示例

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const result = await client.chatAi.processDocument({
  modelName: "gpt-4o",
  url: "https://example.com/blank-invoice.pdf",
  organizationId: "org_xxx",
  fileUrlToFill: "https://example.com/blank-invoice.pdf",
  language: "en",
});

if (result.success && result.data.filledPdfUrl) {
  console.log("已填充的 PDF：", result.data.filledPdfUrl);
}
```
</TabItem>
<TabItem label="Python">
```python
result = client.chat_ai.process_document(
    model_name="gpt-4o",
    url="https://example.com/blank-invoice.pdf",
    organization_id="org_xxx",
    file_url_to_fill="https://example.com/blank-invoice.pdf",
    language="en",
)

if result["success"] and result["data"].get("filledPdfUrl"):
    print("已填充的 PDF：", result["data"]["filledPdfUrl"])
```

## `client.documentAi` — 完整创建（编排器）

端到端文档 AI 代理创建（iMBRACE webapp 所做的）：创建带有提取模式的看板，然后创建 UseCase + 链接到该看板的 AI 代理。

<Tabs syncKey="lang">
<TabItem label="TypeScript">
```typescript
const result = await client.documentAi.createFull({
  name: "发票提取器",
  instructions: "提取发票字段。日期格式为 YYYY-MM-DD。",
  schemaFields: [
    { name: "invoice_number", type: "ShortText", description: "发票 ID" },
    { name: "total",          type: "Number",    description: "总金额" },
    { name: "date",           type: "Date",      description: "发票日期" },
  ],
  modelId: "gpt-4o",
  providerId: "system",
});

console.log(result.board_id);     // "brd_xxx"
console.log(result.ai_agent_id);  // 创建的 AI 代理的 UUID
console.log(result.usecase_id);   // 创建的 UseCase 的 UUID
```
</TabItem>
<TabItem label="Python">
```python
result = client.document_ai.create_full(
    name="发票提取器",
    instructions="提取发票字段。日期格式为 YYYY-MM-DD。",
    schema_fields=[
        {"name": "invoice_number", "type": "ShortText", "description": "发票 ID"},
        {"name": "total",          "type": "Number",    "description": "总金额"},
        {"name": "date",           "type": "Date",      "description": "发票日期"},
    ],
    model_id="gpt-4o",
    provider_id="system",
)

print(result["board_id"])      # "brd_xxx"
print(result["ai_agent_id"])   # 创建的 AI 代理的 UUID
```
</TabItem>
</Tabs>

## 异步用法（Python）

<Tabs syncKey="lang">
<TabItem label="AsyncImbraceClient">
```python
from imbrace import AsyncImbraceClient

async with AsyncImbraceClient() as client:
    # 直接处理（chat_ai）
    models = await client.chat_ai.list_document_models()
    result = await client.chat_ai.process_document(
        model_name="gpt-4o",
        url="https://example.com/invoice.pdf",
        organization_id="org_xxx",
    )

    # 基于代理的处理（document_ai）
    agents = await client.document_ai.list_agents(document_ai_only=True)
    result2 = await client.document_ai.process(
        agent_id=agents[0]["_id"],
        url="https://example.com/receipt.pdf",
        organization_id="org_xxx",
    )
```
