## `AuthError: Invalid or expired access token.` (HTTP 401)

Credential expired or invalid. Generate a new one (see [Setup Guide → Get an API Key](/getting-started/setup/#get-an-api-key)) and update `.env`. For the credential model itself, see [Authentication](/typescript/authentication/) and [Error Handling → AuthError](/typescript/error-handling/#autherror).

```bash
# Update .env
IMBRACE_API_KEY=new_api_key
```

---

## `ApiError: [400] {"message":"must have required property 'type'"}`

A required parameter is missing. Check the resource reference for required fields — see [Resources](/typescript/resources/).

```python
# Incorrect
client.channel.list()

# Correct
client.channel.list(type="web")
```

---

## `ApiError: [404]` with double path in URL

URL is doubled because `IMBRACE_BASE_URL` was incorrectly set to a full endpoint. The correct values are listed in [Local Testing → Environment URLs](/typescript/local-testing/#environment-urls).

```env
# Incorrect
IMBRACE_BASE_URL=https://app-gatewayv2.imbrace.co/private/backend/v1/third_party_token

# Correct
IMBRACE_BASE_URL=https://app-gatewayv2.imbrace.co
```
